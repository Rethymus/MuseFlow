## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-08 | Denial of Service | Auto-save race condition on chapter switch | mitigate | forceSave cancels debounce timer before flushing. _switchChapter awaits _forceSaveCurrent before loading new document. Sequential execution prevents overlap. |
| T-11-09 | Information Disclosure | Chapter content in Hive | accept | Local-only storage per project constraints. Hive boxes are local files. No network transmission. |
| T-11-10 | Denial of Service | SuperEditor Document swap loses state | mitigate | ValueKey(chapterId) on SuperEditor ensures clean document replacement. EditorHolderNotifier updated to new editor on switch (W1 fix). Undo history is per-document (acceptable loss per chapter boundary). |
| T-11-15 | Tampering | Keyboard shortcut collision with system/editor shortcuts | mitigate | Ctrl+Up/Down/N are not used by SuperEditor's default key bindings. SuperEditor uses Ctrl+B/I/U for formatting, Ctrl+Z/Y for undo/redo. No collision. |
</threat_model>
