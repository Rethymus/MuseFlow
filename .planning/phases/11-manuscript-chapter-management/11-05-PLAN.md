<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Chapter context injection | Adjacent chapter content injected into AI prompt |
| Export content | Chapter content serialized for file output |
| Purge service | Cascading delete of manuscript + chapter data |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-11 | Information Disclosure | Chapter context in AI prompts | accept | Only chapter summaries (not full content) injected. Local-only AI calls. No network interception risk for local LLM. |
| T-11-12 | Denial of Service | Purge service on startup | mitigate | Wrapped in try-catch. Purge failure does not prevent app startup. 30-day retention prevents accidental data loss. |
| T-11-13 | Tampering | Export file content | accept | Export writes to user-selected path. Content is user's own data. No cross-user concern in single-user local app. |
</threat_model>
