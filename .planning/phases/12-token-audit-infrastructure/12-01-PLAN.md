## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | token_audit Hive box | accept | Token counts are not security-sensitive. No financial data in this phase (cost deferred to Phase 16). Local-only storage. |
| T-12-02 | Spoofing | API usage response | accept | Usage data from API is used for display only. Fallback estimation exists. Tampered values only affect statistics display. |
| T-12-03 | Input Validation | TokenAuditRecord construction | mitigate | Validate inputTokens >= 0 and outputTokens >= 0 in constructor. Throw ArgumentError on negative values. Validate non-empty manuscriptId. |
| T-12-04 | Denial of Service | enforceLimit cleanup | accept | 10000 record limit with batched cleanup. Worst case: brief I/O spike during cleanup. Not security-sensitive. |
| T-12-SC | Tampering | npm/pip/cargo installs | accept | No new packages installed this phase -- all dependencies already in project. |
