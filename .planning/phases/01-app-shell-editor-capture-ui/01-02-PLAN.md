    Toolbar layout (per D-06, 6 controls):
    1. Bold IconButton (Icons.format_bold, tooltip: '加粗 (Ctrl+B)')

    Bold/italic toggle logic (per RESEARCH Pattern 3):
    - If composer.selection is collapsed: toggle composer.preferences for that attribution (affects new text)
    - If composer.selection is expanded: execute ToggleTextAttributionsRequest with the selection range

    Heading conversion logic:
    - Get nodeId from composer.selection.base.nodeId
    - Get current node via editor.document.getNodeById(nodeId)
    - If node is TextNode: create new ParagraphNode with same id, same text, metadata: {'blockType': selectedHeaderAttribution}
    - Execute ReplaceNodeRequest

    List conversion logic:
    - Same node lookup as heading
    - Create ListItemNode with same id, same text, itemType: ListItemType.unordered or ListItemType.ordered
    - Execute ReplaceNodeRequest

    Toolbar button active state (per RESEARCH Pitfall 2):

    - For italic: same pattern with italicsAttribution
    - Active buttons change color to colorScheme.primary (accent)
    - For heading: check the current node's metadata blockType against header attributions
    - For lists: check if current node is ListItemNode and its itemType
