|----------|-------------|
| App startup -> Hive storage | Initialization sequence opens encrypted/non-encrypted boxes |
| App startup -> flutter_secure_storage | API key storage delegates to Windows Credential Manager |
| Window events -> Settings persistence | Window resize/move events trigger async writes to Hive |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Information Disclosure | API key storage | mitigate | flutter_secure_storage with Windows Credential Manager backend -- keys never in plaintext |
| T-01-02 | Information Disclosure | Settings box | mitigate | Hive encrypted box with AES cipher for settings containing window preferences |
| T-01-03 | Tampering | Fragment data | accept | Fragments are user creative content, stored locally, no integrity risk beyond user action |
| T-01-SC | Tampering | Flutter/Dart packages | mitigate | All packages verified in Phase 0 pub cache, no new packages installed in this plan |
