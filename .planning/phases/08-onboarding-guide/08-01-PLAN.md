## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Tampering | onboarding_completed flag in Hive | accept | Local-only app, no security impact if user manually clears flag in local DB |
| T-08-02 | Denial of Service | Malformed OnboardingProgress JSON crashes redirect | mitigate | JSON parse in fromJson wrapped in try-catch, default to OnboardingProgress.initial() on error |
| T-08-03 | Tampering | onboarding_progress corrupted state | mitigate | getProgress() returns default OnboardingProgress if null or invalid, prevents null crashes |
</threat_model>
