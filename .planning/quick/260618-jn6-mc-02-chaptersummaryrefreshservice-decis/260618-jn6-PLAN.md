| Local Hive box → Service | `chapter_summary_repository.put/getByChapterId` reads user-generated JSON from local Hive storage; corrupted/legacy entries throw StateError |
| Active AI provider config → Service | `activeApiKeyProvider`/`activeProviderProvider` are user-configured values from secure storage + provider management UI; null when unconfigured |
| LLM network stream → Service | `summarizationService.summarize` calls adapter.createStream which may throw AIStreamException/AIRateLimitException/AIAuthException on transient or auth failures (wma retry throat) |
| save() main contract ↔ fire-and-forget refresh | The TRIGGER boundary: AIException/StateError from refresh MUST NOT propagate into ChapterNotifier.save |
