## Design

```dart
// 新增数据集（line ~550 后，_formulaicEndings 之前）
static const List<String> _wuxiaCliches = [
  '内力运转', '施展轻功', '剑光一闪', '刀光剑影',
  '真气鼓荡', '身法如电', '招式凌厉', '武学修为',
];

// _buildReviewSignals genre block 改造（line 854-866）：
final xianxiaHits = _countPhraseHits(text, _xianxiaCliches);
final wuxiaHits = _countPhraseHits(text, _wuxiaCliches);
final genreCount = xianxiaHits + wuxiaHits;          // 合计阈值不变(>=2)
if (genreCount >= 2) {
  final genre = xianxiaHits >= wuxiaHits ? '修仙' : '武侠';  // 主导命名
  signals.add(ReviewSignal(
    title: '类型文套句偏多',                            // 不变
    description: '$genre常见短语重复出现，可能削弱作者自己的画面感。',
    severity: genreCount >= 4 ? high : medium,
    evidence: '$genreCount 次',
  ));
}
```

向后兼容：纯修仙文本 xianxiaHits>=2>wuxiaHits→"修仙"（同现状）；纯武侠文本此前 genreCount=0 无信号→现 wuxiaHits>=2 有"武侠"信号（**新覆盖**，正向）。
