# MuseFlow 编辑器模块使用指南

## 快速开始

### 1. 基础使用

编辑器提供了简洁直观的界面，包含以下主要区域：

**主编辑区**
- 中央的大文本输入区域
- 支持多行文本输入
- 支持文本选择和基本编辑操作

**侧边栏（思维碎片）**
- 左侧显示所有思维碎片
- 点击"+"按钮添加新碎片
- 点击碎片可将其内容插入到主编辑区
- 长按碎片进行编辑或删除

**工具栏（顶部）**
- 切换侧边栏显示/隐藏
- 撤销/重做操作
- 格式清洗按钮
- 上下文锚点设置

**AI操作栏（底部）**
- 润色：改善选中文本的表述
- 扩写：扩展选中文本的内容
- 大纲：为全文生成结构化大纲

### 2. 思维碎片功能

#### 添加碎片
1. 点击侧边栏的"+"按钮
2. 在弹出的对话框中输入你的想法
3. 可选：添加标签来分类管理
4. 点击"添加"按钮保存

#### 使用碎片
- **插入到编辑区**：点击碎片卡片，内容会插入到光标位置
- **编辑碎片**：点击碎片卡片的编辑图标
- **删除碎片**：点击碎片卡片的删除图标

#### 碎片管理
- 碎片按创建时间排序
- 显示相对时间（如"5分钟前"）
- 支持标签分类显示
- 可选择多个碎片批量操作

### 3. AI辅助写作

#### 文本润色 (Ctrl+K)
1. 选中需要润色的文本
2. 按下 Ctrl+K 或点击"润色"按钮
3. AI会优化文本的表达和流畅度
4. 润色后的文本会替换原选中文本

**示例：**
```
原文本：这个想法很好，但是需要更多细节
润色后：这一观点颇具价值，若能补充更多具体细节将更加完善。
```

#### 文本扩写 (Ctrl+E)
1. 选中需要扩写的文本
2. 按下 Ctrl+E 或点击"扩写"按钮
3. AI会基于原文扩展更多内容
4. 扩写内容会插入到选中文本之后

**示例：**
```
原文本：人工智能正在改变教育
扩写后：人工智能正在深刻地改变教育领域。从个性化学习到智能辅导，从自动评估到课程推荐，AI技术在教育中的应用日益广泛，为学习者提供了更加精准和高效的学习体验。
```

#### 生成大纲 (Ctrl+O)
1. 输入完整的文章内容
2. 按下 Ctrl+O 或点击"大纲"按钮
3. AI会分析内容并生成结构化大纲
4. 大纲会以对话框形式展示

**示例：**
```
1. 引言
   - 人工智能的发展背景
   - 教育领域的重要性

2. AI在教育中的应用
   - 个性化学习
   - 智能辅导系统

3. 挑战与展望
   - 技术局限性
   - 未来发展方向
```

### 4. 上下文锚点

上下文锚点功能可以让AI更好地理解你的写作意图：

#### 设置锚点
1. 选中要作为上下文的文本
2. 点击工具栏的锚点图标
3. 锚点文本会显示在底部AI栏中

#### 使用锚点
设置锚点后，所有的AI操作都会参考这个上下文，提供更准确的结果。

**示例：**
```
锚点文本：本文讨论人工智能在医疗领域的应用
润色文本：AI能帮助医生诊断
结果：人工智能技术能够有效协助医师进行疾病诊断，提高诊断准确性。
```

### 5. 格式清洗

格式清洗功能可以清除不需要的Markdown残留：

#### 清洗模式
- **完整清洗**：清除所有Markdown格式，保留纯文本
- **智能清洗**：保留基本结构（标题、列表），清除内联格式
- **轻量清洗**：只清除明显的格式问题（多余空行、空格）

#### 使用方法
1. 点击工具栏的"格式清洗"按钮
2. 系统自动应用清洗规则
3. 显示清洗结果和影响的格式数量

### 6. 快捷键大全

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Ctrl+K | AI润色 | 润色选中文本 |
| Ctrl+E | AI扩写 | 扩展选中文本 |
| Ctrl+O | 生成大纲 | 为全文生成大纲 |
| Ctrl+Z | 撤销 | 撤销上一步操作 |
| Ctrl+Y | 重做 | 重做已撤销的操作 |
| Ctrl+A | 全选 | 选择全部文本 |
| Ctrl+C | 复制 | 复制选中文本 |
| Ctrl+V | 粘贴 | 粘贴剪贴板内容 |
| Ctrl+X | 剪切 | 剪切选中文本 |

## 高级技巧

### 1. 写作工作流

**灵感收集阶段**
1. 快速记录想法为思维碎片
2. 使用标签进行分类
3. 不关心格式，专注内容

**初稿写作阶段**
1. 将碎片拖入主编辑区
2. 快速组织内容结构
3. 使用AI扩写丰富内容

**润色修改阶段**
1. 使用上下文锚点设置写作风格
2. 分段润色改善表达
3. 生成大纲检查逻辑

**定稿发布阶段**
1. 格式清洗统一风格
2. 生成摘要用于预览
3. 导出最终版本

### 2. AI操作最佳实践

**润色建议**
- 一次只润色一个段落
- 设置相关的上下文锚点
- 避免过度润色失去个人风格

**扩写建议**
- 扩写前明确扩展方向
- 使用具体主题而非宽泛概念
- 扩写后人工调整保持一致性

**大纲建议**
- 用于检查文章结构
- 验证逻辑流程
- 发现缺失的内容

### 3. 思维碎片管理

**分类策略**
- 按主题添加标签
- 按状态区分（想法/草稿/完成）
- 按优先级排序

**使用技巧**
- 每日整理碎片内容
- 定期清理无用的碎片
- 将相关碎片关联起来

## 故障排除

### 常见问题

**Q: AI操作没有响应怎么办？**
A: 检查网络连接，确认AI服务配置正确，查看错误提示信息。

**Q: 格式清洗后格式不对？**
A: 使用"智能清洗"模式保留基本结构，或手动调整。

**Q: 快捷键不起作用？**
A: 确认焦点在编辑区，检查是否有快捷键冲突。

**Q: 思维碎片丢失？**
A: 检查是否被误删除，查看数据存储设置。

### 性能优化

**处理大文本**
- 分段处理而非一次性处理
- 避免频繁的AI操作
- 定期清理历史记录

**内存管理**
- 关闭不需要的碎片
- 清理编辑历史
- 重启应用释放内存
