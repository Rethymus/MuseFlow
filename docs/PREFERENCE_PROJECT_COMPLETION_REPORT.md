## 🏗️ 技术架构

### 分层架构
```
UI Layer (页面和组件)
    ↓
Service Layer (业务逻辑)
    ↓
Model Layer (数据模型)
    ↓
Storage Layer (本地存储)
```

### 核心算法
- **指数移动平均 (EMA)**: 动态更新接受率
- **Sigmoid函数**: 置信度计算
- **Jaccard相似度**: 文本相似度计算
- **TF-IDF简化**: 关键词提取
- **多维度分析**: 风格和结构检测

### 数据流
```
用户输入 → 写作分析 → 偏好更新 → AI调整 → 用户反馈 → 学习优化
```
