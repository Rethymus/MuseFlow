# 第68章 身世之谜

夜色深沉，青云宗后山禁地之中，清虚真人闭目端坐，身周，如同海浪翻腾。他面前，林风恭立，眼神中闪烁着好奇与敬畏。
“林风，你可知，你体内的无名功法，其源头何在？”清虚真人的声音低沉，仿佛自远处的山涧传来，带着一股不可抗拒的力量。
林风，他深知这个问题非同小可，他缓缓摇头：“弟子不知，但自从修习此功法以来，似乎每前进一步，便有新的感悟，却又始终无法窥其全貌。”
清虚真人睁开眼，直视林风：“那是因为，这功法并非凡间之物，而是上古时期，你师门遗失的绝学。”
林风心头一震，师门？他想起那块上古玉简，那仿佛是开启记忆之门的钥匙。他小心翼翼地问道：“我的师门？弟子一直以为，我父母双亡，是孤儿。”
清虚真人缓缓摇头，语气中带着一丝叹息：“非也，非也。你的父母，并非凡人，而是我青云宗的先辈，他们为了守护宗门，不惜牺牲自己，将你留在了世间。”
林风，父母？他从未想过，自己竟然有这样的身世。他想要上前，却又不敢打扰清虚真人，只觉得喉咙一阵干涩。
“你的父母，曾是青云宗的顶尖高手，他们曾与我一同修炼一门名为‘灵虚诀’的秘法。”清虚真人继续说道，“而这无名功法，便是‘灵虚诀’的传承。”
林风听到这里，心中掀起了惊涛骇浪。他修炼的功法，竟然与自己的身世有着如此密切的联系。他不禁问道：“那为何我从未听说过这些？”
清虚真人深深地看了林风一眼，语气中带着一丝沉痛：“因为，你的父母在临终前，将一切秘密都封印在了你的记忆之中，只待有缘人唤醒。”
林风心头一震，原来，自己一直以来的迷茫，原来都是因为这份封印。他心中涌起一股强烈的责任感，他想要揭开这个秘密，揭开自己的身世之谜。
“清虚真人，弟子想了解这一切。”林风的声音坚定而有力。
清虚真人点了点头，欣慰：“好，从今往后，你便是‘灵虚诀’的传承者，你的使命，便是揭开这份封印，找回你父母的记忆，恢复‘灵虚诀’的传承。”
林风心中一动，他明白了，自己的道路，将不再平凡。他将肩负起这个使命，揭开身世之谜，寻找自己的父母，恢复‘灵虚诀’的传承。
夜色愈发浓重，林风站在清虚真人面前，感受着灵气在体内涌动，他闭上眼睛，感受着这份传承的力量。他知道，从今往后，他的道路，将充满未知与挑战。
而在这禁地的深处，月光下的清虚真人，眼中也闪过一丝坚定。他看着林风，心中默默祈祷，愿这传承，能带给林风力量，让他走得更远。
林风缓缓睁开眼，他感受到一股新的力量在体内涌动，那是传承的力量，那是父母留给他的力量。他抬起头，目光坚定，他知道自己，将踏上新的征程。
他转身，朝着禁地的深处走去，每一步都显得沉稳而有力。他知道，前方，将有无数的挑战，但他已无所畏惧。
月光洒在他的身上，如同披上了一层圣洁的光辉。林风心中默念，无论前路如何，他都将勇往直前，追寻自己的身世之谜，追寻那失落已久的传承。林风的脚步声在寂静的禁地中回响，他每走一步，都似乎在接近一个全新的世界。月光如水，洒在他身上，也洒在他心中那些未曾触碰的秘密之上。
身后，清虚真人静静地看着他，眼中满是期待。他明白，林风这一去，不仅是为了追寻身世之谜，更是为了承担起那份沉甸甸的责任。
“林风，记住，‘灵虚诀’不仅是一门功法，它更是一种信念，一种传承。”清虚真人低沉的声音在林风的耳边回荡。
林风停下脚步，转身面向清虚真人，目光中满是虔诚：“弟子铭记在心，无论是何种挑战，弟子都将坚持到底。”
清虚真人微微点头，从袖中取出一枚古朴的玉佩，递给林风：“这是你父母的遗物，也是你身份的证明。它能够帮助你解开一些谜团，但更多的是，它将引导你走上正确的道路。”
林风接过玉佩，入手冰凉，他感觉到一股熟悉的力量从玉佩中散发出来，那是一种让他心安的力量。
“清虚真人，弟子还有一个疑问。”林风犹豫了一下，终于鼓起勇气问道。
“你说。”
“为何要等到现在才告诉我这些？”
清虚真人沉默片刻，：“因为时机未到。你尚未准备好，你的实力还不够强大，你的心性还不够成熟。但现在，你有了足够的理由，也有了足够的动力。”
林风微微颔首，他明白清虚真人的苦心。他转身，向着禁地的深处走去，每一步都坚定而有力。
禁地深处，灵气更加浓郁，仿佛有无形的手在引导着林风。他闭上眼睛，用心去感受这份灵气，去感受那份传承的力量。
突然，他的眼前出现了一片光幕，那光幕中，是他父母的身影，他们在与敌人战斗，他们在守护着青云宗，他们在坚守着那份信念。
“林风，你的父母，他们是英雄，他们是榜样。”清虚真人的声音在林风的耳边响起，“他们的牺牲，没有白费，你将是他们精神的延续。”
林风的眼中涌上了泪水，他感受到一股强烈的情感，那是属于他的血脉，那是属于他的责任。
“弟子明白。”林风的声音坚定而有力。
光幕逐渐消失，林风睁开眼睛，他的眼中闪烁着决然的光芒。他知道，自己的道路，已经不再是追求个人的修仙之路，而是承载着父母期望和青云宗荣誉的使命之路。
他，感受着体内那股新生的力量，他知道，那力量来源于他的血脉，来源于他的传承。
他转身，向着禁地的深处走去，每一步都如同踏在了坚实的土地上。他不再迷茫，不再困惑，他的心中只有坚定，只有信念。
月光洒在他的身上，如同为他披上了一层金色的披风。林风心中默念，无论前路如何，他都将勇往直前，追寻自己的身世之谜，追寻那失落已久的传承。
身后，清虚真人静静地站在原地，他的眼中满是欣慰。他知道，林风已经做好了准备，他已经踏上了属于自己的征程。
夜色浓重，禁地的深处，林风的身影逐渐消失在月色中。他的脚步声，虽然微弱，但却坚定，如同在诉说着一个全新的故事。林风的身影在禁地深处渐行渐远，他的步伐虽然沉重，但每一步都踏得坚实。月光如流水般洒在他的身上，映照出他坚毅的面容。
就在他即将走出禁地的那一刻，一道刺眼的光芒突然从禁地深处爆发，瞬间照亮了整个禁地。林风停下脚步，抬头望去，只见那光芒中，一个巨大的虚影缓缓浮现，那是一个身穿古老战甲，手持长剑的男子，他的眼神冷冽，仿佛能洞察一切。
“这是……”林风心中一惊，他认出了那虚影，那是他父母战斗时的样子，是他从未见过的英勇。
“林风，你父母的真身。”清虚真人的声音在林风的耳边响起，他的声音中充满了敬仰。
林风闭上眼睛，感受着那虚影中的力量，那是一种他从未体验过的强大。他明白，那力量来自于他的血脉，来自于他的传承。
“弟子愿继承父母的意志，守护青云宗，守护修仙界。”林风的声音坚定而有力。
虚影缓缓消散，光芒也逐渐消失，但林风却感受到了一股强大的力量涌入他的体内。那力量如同潮水般涌动，他的身体仿佛要被撕裂。
“林风，你承受得住吗？”清虚真人担忧地问道。
林风咬紧牙关，他感受到体内的力量在疯狂地涌动，但他并没有感到痛苦，反而感到一种前所未有的畅快。
“弟子可以。”林风的声音中充满了自信。
，禁地深处突然传来一阵剧烈的震动，仿佛有什么东西正在苏醒。林风心中一惊，他转身望去，只见禁地的深处，一道裂缝缓缓裂开，一股强大的气息从中散发出来。
“那是……”林风心中一惊，他认出了那股气息，那是上古大能的气息。
“那是封印。”清虚真人，“那封印封印着的是天衡盘，它蕴含着修仙界的至高力量。”
林风坚定：“弟子愿意承担起这份责任，守护天衡盘。”
清虚真人微微点头，他明白林风已经做好了准备。他转身，从袖中取出一枚古朴的玉简，递给林风：“这是天衡盘的传承，你若能解开它的封印，你将获得修仙界的至高力量。”
林风接过玉简，感受着玉简中的力量，他的心中充满了期待。
，禁地深处的震动越来越剧烈，仿佛有什么东西正在挣扎着苏醒。林风心中一惊，他转身，朝着禁地深处走去。
“林风，小心。”清虚真人提醒道。
林风点头，他的心中充满了决心。他知道自己，已经踏上了新的征程，他必须面对所有的挑战，他必须解开身世之谜，他必须守护修仙界。
禁地深处，林风感受到一股强大的力量在涌动，那力量仿佛要吞噬一切。他闭上眼睛，感受着那力量，他的心中充满了恐惧，但他并没有退缩。

林风睁开眼睛，他的眼中充满了坚定：“弟子明白，弟子会继续努力。”
，禁地深处的裂缝突然扩大，一股强大的气息从中爆发出来。林风心中一惊，他转身，朝着裂缝走去。
“林风，小心。”清虚真人提醒道。
林风点头，他的心中充满了决心。他知道自己，已经踏上了新的征程，他必须面对所有的挑战，他必须解开身世之谜，他必须守护修仙界。
