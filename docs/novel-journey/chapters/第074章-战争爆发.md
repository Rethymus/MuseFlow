# 第74章 战争爆发

乌云翻滚，狂风肆虐，青云宗的南大门被一片黑暗笼罩。战鼓声声，如雷贯耳，伴随着远处隐约传来的嘶吼与咆哮，让人不寒而栗。
清虚真人站在城墙上，凝视着远方。他身着一袭青衣，须发皆白，却难掩那股威严与深邃。他身旁，林风紧握着手中的长剑，剑身闪烁着寒光，与他的眼神相映成辉。
“清虚真人，暗影门联合了数个邪派，攻势猛烈，我们该如何应对？”林风的声音低沉而坚定。
清虚真人没有回答，只是微微点头，目光扫过城墙上的守军。他看到了那些年轻的身影，他们或紧张，或兴奋，但无一例外，都紧握着武器，准备迎接这场生死之战。
“林风，你负责守卫南大门，务必坚守。”清虚真人语气严肃。
林风点头，心中涌起一股责任感。他，感受着体内涌动的剑意，一股强大的力量在体内流转。
城墙下，黑压压的敌人如同潮水般涌来。他们手中拿着各种武器，有的血腥，有的诡异，但无一例外，都充满了杀气。
“林风，你注意到了吗？”苏雪晴的声音从身后传来，她的眼神中充满了担忧。
林风回头，看着苏雪晴，只见她身着一袭白裙，长发飘飘，脸上带着担忧之色。他：“没有，大师姐，我在集中精神。”
苏雪晴，没有再说什么。她知道，林风已经进入了一种境界，他现在只关注着前方。
城墙上的守军开始反击，箭矢如雨，刀剑交击。林风站在最前方，他的剑法凌厉，每一次挥剑都带走一个敌人的生命。
“林风，小心！”赵天磊的声音从侧后方传来，他手中长枪舞动，挡住了从侧翼偷袭的敌人。
林风回头，看着赵天磊，微笑着点了点头。他知道，赵天磊已经成为了他最坚实的后盾。
战斗越来越激烈，敌人的攻势愈发猛烈。林风心中焦急，他不能让清虚真人失望，不能让青云宗的弟子们失望。
“林风，你的剑法越来越纯熟了。”清虚真人的声音从远处传来，带着一丝赞许。
林风心中一暖，他知道，清虚真人已经看到了他的努力。
“清虚真人，我会守护好青云宗的。”林风的声音坚定而有力。
“好。”清虚真人应了一声，目光再次扫过战场，他的眼神中充满了决心。
城墙下，敌人如同潮水般涌来，但林风和他的同伴们却如同磐石一般，坚守着南大门。
“林风，你的剑法越来越纯熟了。”苏雪晴的声音再次传来，她的声音中带着一丝惊喜。
林风回头，看着苏雪晴，微笑着点了点头。他知道，他的努力没有白费。
“大师姐，我会守护好青云宗的。”林风的声音坚定而有力。
“好。”苏雪晴应了一声，她的眼神中充满了信任。
