# 第44章 重伤昏迷

昏沉的疼痛如同潮水般涌来，林风的眼眸紧闭，额头渗出细密的汗珠。他躺在一间幽静的密室内，四周的墙壁上镶嵌着一块块闪烁着淡淡光芒的灵石，空气中弥漫着淡淡的灵气，让人感到一丝安宁。
密室的门被轻轻推开，苏雪晴的身影出现在门口，她的眼神中充满了担忧和焦虑。她轻手轻脚地走进来，走到床边，坐在床沿上，双手轻轻握住林风的手，掌心传来的温暖让林风微微一颤。
“大师姐……”林风的嘴唇轻启，声音低哑而微弱。
苏雪晴，眼神中闪过一丝不易察觉的痛苦：“风儿，你醒醒，别怕，我在这里。”
她的声音轻柔，如同春风拂过，却无法驱散林风心中的阴霾。她的眼神扫过林风身上缠绕的伤口，那些伤口已经化作了暗红色，显然是凝丹失败后的余伤。
苏雪晴站起身，走到一旁的药架前，她的动作轻缓而熟练，取出一瓶丹药，小心翼翼地倒出几粒，用灵力包裹着，然后缓缓走到林风床边，将丹药喂入他的口中。
“风儿，你一定要坚强，我知道你不会轻易放弃的。”苏雪晴的声音带着一丝颤抖，她将手轻轻放在林风的额头，感受着他的温度。
白灵蜷缩在床边，它的眼神中充满了关切，不时地用小爪子轻轻触碰林风的肩膀，用灵力为林风疏通经脉。它的毛发微微泛着光，仿佛在用自己的力量为林风疗伤。
“大师姐，白灵……”林风的声音再次响起，虽然微弱，却充满了力量。
苏雪晴回头，看到白灵的样子，欣慰：“你们都很好，我……我知道。”
她的声音再次变得哽咽，她深深地吸了一口气，调整了一下情绪，然后重新坐回床边，握住林风的手，闭上眼睛，开始默念着疗伤的咒语。
她的手指轻轻地在林风的手背上滑动，每一次触摸都仿佛带着一股暖流，缓缓流入林风的体内。
“清虚真人……”林风的声音再次响起，他的气息更加微弱。
苏雪晴睁开眼睛，她的眼神中闪过一丝慌乱，她知道清虚真人出手护住林风的心脉，但若是经脉无法恢复，林风的修为可能会倒退。
她起身，快步走出密室，来到院落中，焦急地呼唤着清虚真人。
“清虚真人，林风他……”
清虚真人身影出现在院落中，他的眼神严肃，脸上没有一丝表情，但他的手中却已经凝聚出一道道灵力，准备随时出手。
“真人，林风的经脉……”苏雪晴的声音带着一丝哭腔。
清虚真人没有说话，他的目光落在林风身上，凝重，然后他伸出右手，掌心向天，一道道灵力从天而降，汇聚在他的掌心。
“风儿，我尽力了。”清虚真人的声音低沉，充满了力量。
他的掌心开始发光，那道光芒如同太阳般耀眼，瞬间笼罩了林风。
“若……若无法恢复，修为……”林风的声音再次微弱地响起。
清虚真人没有回答，他的眼神坚定，仿佛在告诉林风，无论结果如何，他都会尽力守护。
苏雪晴看着这一幕，心中充满了不安，她知道清虚真人是在用自己的力量为林风护住心脉，但若是经脉无法恢复，林风的未来将充满未知。
白灵静静地蜷缩在床边，它的眼神中充满了坚定，仿佛在告诉林风，无论发生什么，它都会陪伴在他的身边。
林风躺在床上，他的眼神紧闭，他的身体在灵力的包裹下，开始蠕动，每一次蠕动都仿佛在向世界宣告，他不会轻易放弃。
而这一切，都在这密室之中，静静地发生着。苏雪晴站在清虚真人身后，她的眼神紧盯着林风，每一次灵力的流转，每一次光芒的闪烁，都牵动着她的心。她知道，清虚真人出手，是林风的希望。
“真人，能否告诉我，林风的经脉……恢复的几率有多大？”苏雪晴的声音颤抖，她的心中充满了不安和恐惧。
清虚真人没有回头，他的目光依旧集中在林风身上，他的声音低沉而坚定：“雪晴，我尽力而为，但若真的无法恢复，他的修为可能会倒退。”
苏雪晴的心沉了下去，她知道这意味着什么。修为倒退，对于林风这样的修仙者来说，几乎等同于从头开始。
