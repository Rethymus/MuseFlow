# 第65章 王磊的下场

藏经阁顶层，阳光透过云层洒下，照亮了那座古老的玉简。林风站在玉简前，双手微抬，感受着那股古老而强大的力量涌入自己的经脉。
他的，心中：“这股力量...似乎是剑意！”他的心中闪过一丝惊讶，他从未接触过剑意，但此刻，他能清晰地感受到那股剑意在自己体内的流动。
，藏经阁大门被推开，清虚真人走了进来。他的眼神锐利如鹰，直接看向林风。
“林风，你最近似乎有些不同。”清虚真人的声音低沉而有力。
林风一愣，然后：“真人，我只是在修习功法时，感悟到了一丝剑意。”
清虚真人点了点头，赞许：“很好，剑意对于修仙者来说，是一种强大的辅助力量。你现在的修为虽然不高，但能感悟到剑意，已是不易。”
说完，清虚真人走到林风身旁，目光落在玉简上：“这玉简，你还有保存着吗？”
林风点了点头，从怀中取出玉简：“真人，我本想自己研究，但现在看来，我可能需要你的指导。”
清虚真人接过玉简，目光在玉简上扫过，然后微微一叹：“这玉简上的功法，乃是我青云宗历代宗主所修，名为‘苍穹剑诀’。此剑诀威力强大，但修炼起来却艰难。你既然能感悟到剑意，或许有资格修炼此剑诀。”
林风心中一喜，他一直在寻找一个合适的功法来突破自己的瓶颈，如今清虚真人主动提出，他自然是求之不得。
，藏经阁外突然传来一阵喧哗。林风和清虚真人对视一眼，都看出了彼此眼中的疑惑。
“怎么回事？”清虚真人沉声问道。
林风走出藏经阁，只见一群人正围在广场上，指着一个人指指点点。那个人正是王磊。
“王磊，你竟然勾结暗影门！”人群中有人指着王磊，愤怒地说道。
王磊瞪大了眼睛，不可置信地看着周围的人群。他的脸色铁青，狠厉。
“你们...你们误会了！我...我并没有勾结暗影门！”王磊声嘶力竭地辩解道。
，他的话音未落，人群中立刻有人拿出证据。
“王磊，你还在狡辩！这是你与暗影门的交易记录！”一个人拿出一张纸，上面清楚地写着王磊与暗影门交易的细节。
王磊脸色瞬间惨白，他知道自己的罪行已经败露。
“长老们，我...我真的没有勾结暗影门！”王磊跪在地上，声泪俱下。
，他的辩解已经无人理会。长老们纷纷站了出来，目光冰冷地盯着王磊。
“王磊，你背叛青云宗，勾结邪教，罪不容诛！”一位长老。
王磊脸色惨白，身体剧烈颤抖。他知道，自己的结局已经注定了。
“林风，你还有何话说？”长老们看向林风，期待着他的表态。
林风沉默了片刻，然后缓缓开口：“长老们，我认为王磊虽然罪有应得，但他还有家人，我们应该给他一个机会，让他和家人团聚后再行处决。”
长老们相视一眼，点头同意。
，赵天磊走了过来。他的眼神中充满了恨意，直盯着王磊：“林风，你为什么要帮他？”
林风：“赵天磊，你为何对我如此恨意？”
赵天磊一愣，然后愤怒地说道：“因为你...因为你抢走了我的一切！”
林风摇头：“赵天磊，你的心胸未免太狭隘了。我们本是同门，应该携手共进，共同进步。何必为了区区恩怨，而断送自己的前程？”
赵天磊冷哼一声，不再理会林风，转身离去。
林风看着赵天磊的背影，叹息。他知道，赵天磊已经深陷仇恨的泥潭，想要让他回头，恐怕已经很难了。
而王磊，则被长老们押走。他的眼神中充满了绝望，他知道，自己的生命已经走到尽头。
林风看着王磊被押走，心中没有丝毫喜悦，反而有些沉重。他知道，王磊的死，只是整个修仙界阴谋的开始。
，清虚真人走了过来，拍了拍林风的肩膀：“林风，你不必太过担心。王磊的死，只是修仙界阴谋的一个环节。我们还有更重要的事情要做。”
林风点了点头，他知道，自己肩负着更重的责任。他需要尽快提升自己的实力，保护身边的人。
