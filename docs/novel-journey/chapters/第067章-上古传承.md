# 第67章 上古传承

幽深的禁地，寒气逼人，洞府深处。苏雪晴盘膝而坐，她体内的剑气缓缓流转，每一次流转都仿佛触动了禁地的秘密。
她的眼前，那把神秘的剑，依旧静静地躺在地上，剑身散发出淡淡的寒光，仿佛在诉说着一个古老的故事。苏雪晴的指尖轻轻触碰到剑身，剑身微微颤动，仿佛回应了她的触碰。
她闭上了眼睛，尝试着与剑沟通，但只有冰冷的金属感在指尖流转，没有任何回应。她的内心充满了疑惑，这座禁地，这把剑，这一切，似乎都与她的身世之谜有关。
，洞府内的灵气突然波动起来，一股强大的威压从深处传来。苏雪晴心中一惊，她知道，这股威压的主人，一定是一位强大的修仙者。
她的眼前，一道身影缓缓出现，那是一个身穿道袍，面容苍老，但眼中却透露出无尽智慧的老者。他的气息强大而深邃，仿佛可以看透一切。
苏雪晴心中一震，她认出了这个老者，他就是青云宗的长老，清虚真人。
“清虚真人？”苏雪晴微微开口，声音中带着一丝敬畏。
清虚真人看着苏雪晴，复杂的情绪，他缓缓开口：“雪晴，你真的准备好了吗？”
苏雪晴心中一凛，她知道，清虚真人问这个问题，一定意味着接下来的事情，非同小可。
“我……我准备好了。”苏雪晴，回答道。
清虚真人点了点头，他的目光落在了地上那把神秘的剑上，怀念。
“这把剑，名为‘苍穹’，是上古时期一位名为‘灵虚真人’的大能所铸，他曾在这座禁地修行，留下了无数的传承。”
苏雪晴心中一动，灵虚真人？这个名字，她似乎在哪里听过。
“而灵虚真人，正是我的前世。”清虚真人淡淡说道，他的声音中带着一丝沉痛。
苏雪晴惊讶地看着清虚真人，她从未想过，这位青云宗的长老，竟然与上古的大能有着如此深的渊源。
“守护青云宗，守护这个传承，是我前世，也是我今生的使命。”清虚真人语气坚定地说道。
苏雪晴沉默了，她看着清虚真人，心中充满了敬佩。
，洞府内的灵气再次波动，一道身影缓缓出现，那是一个身穿黑袍，面容阴冷的男子。他的眼神中充满了仇恨，他的目标，直指苏雪晴。
“苏雪晴，你害得我失去了一切，今天，我要你付出代价！”男子怒吼道。
苏雪晴怒火，她知道，这个男子，就是她一直以来都在躲避的仇人。
“王磊，你敢！”清虚真人怒吼道，他的身体瞬间化为一道流光，瞬间出现在王磊面前。
王磊一惊，他没想到清虚真人会如此迅速地出现在自己面前。
“清虚真人，你……你为什么要保护她？”王磊怒吼道。
清虚真人看着王磊，悲伤：“因为，她是我青云宗的未来，也是我前世的传承。”
王磊瞪大了眼睛，他没想到清虚真人竟然会说出这样的话。
“不，我不信！你一定是在骗我！”王磊怒吼道，他的身体瞬间爆发出强大的气息。
苏雪晴看着王磊，她的眼中充满了坚定：“王磊，你错了，清虚真人说的没错，我是青云宗的未来，也是你无法阻止的未来。”
王磊看着苏雪晴，眼中充满了仇恨，但他却无法动弹，因为他的气息，已经被清虚真人完全压制。
，洞府内的灵气再次波动，一道身影缓缓出现，那是一个身穿白衣，气质清冷的女子。她的眼神中充满了忧虑，她的目光，直指苏雪晴。
“雪晴，你……你真的要这么做吗？”女子微微开口，声音中带着一丝担忧。
苏雪晴看着女子，她的眼中充满了坚定：“雪儿，我知道你担心我，但我必须这么做，为了青云宗，为了我自己的命运。”
女子看着苏雪晴，复杂的情绪，她最终点了点头：“雪晴，我支持你。”
苏雪晴看着女子，她的心中充满了感激，她知道，有了雪儿的支持，她一定能够完成自己的使命。
