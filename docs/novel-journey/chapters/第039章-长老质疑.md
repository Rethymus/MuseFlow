# 第39章 长老质疑

本章《剑道苍穹》第39章 第39章 长老质疑
藏经阁内，灯火辉煌，静谧无声。几位长老正围坐在一张古朴的桌前，落在桌上那卷薄薄的玉简之上。玉简之上，龙飞凤舞，记载着一种神秘莫测的功法。
“这功法，似曾相识。”一位须发皆白的长老眉头紧锁，缓缓开口。
“正是如此，这功法之中，蕴含着强大的灵力，与青云宗的功法截然不同。”另一位长老点头附和，忧虑。
林风站在一旁，面色平静，但心中却是波澜起伏。他知道，这些长老之所以对这功法产生怀疑，是因为这功法并非出自青云宗，而是他偶然所得的上古玉简。
“清虚真人，这功法究竟从何而来？”一位长老沉声问道，语气中带着不容置疑的威严。
清虚真人，站起身来，目光扫过在座长老，：“这功法，是林风在藏经阁中偶然所得，与他个人机缘有关，与青云宗无关。”
“个人机缘？”一位长老冷笑一声，“清虚真人，你可知道，这功法之中蕴含的灵力，若非修为深厚，根本无法领悟？”
“我知道。”清虚真人目光坚定，“林风虽然天资平平，但他意志坚定，心志如铁，正是这等毅力，让他领悟了这功法。”
“可这功法，若是出自外门，岂不是对本宗声誉有所影响？”一位长老语气不善。
“本宗声誉，自然重要。”清虚真人，“但林风若能凭借此功法，突破自我，成为本宗的骄傲，那又何妨？”
话音刚落，便有几道目光落在林风身上，其中不乏审视和质疑。
林风，心中却是坚定。他知道，自己不能被这些质疑所动摇，他要用自己的实力，证明自己的价值。
“清虚真人，我们不是要否定林风的努力。”另一位长老，“只是，这功法来历不明，若是真的如他所言，出自上古玉简，那……”
“那又如何？”清虚真人，“只要林风能将这功法练至巅峰，对本宗有何影响？”
“话是如此。”长老沉吟片刻，终究还是叹了口气，“只是，这功法若真是上古玉简所载，那其中必含有惊天秘密。若是落入他人之手，对本宗，乃至整个修仙界，恐怕都是一场灾难。”
“所以，我们才要暗中调查。”一位长老，“探查这功法的来历，以及林风的修炼秘密。”
清虚真人目光微闪，沉默片刻，终是点头：“也好，此事便交给我来处理。我会保护好林风，不让任何人打扰他的修炼。”
林风站在一旁，目光坚定，心中却是思绪万千。他知道，自己已经站在了风口浪尖，但无论如何，他都要坚持自己的道路，用实力证明自己的价值。
，一阵急促的脚步声传来，一位弟子急匆匆地跑了进来，脸上带着焦急之色。
“长老，不好了，暗影门的人……”
话未说完，便见那位弟子脸色大变，指着窗外，失声道：“他们……他们来了！”
林风目光一凝，瞬间明白，那些长老已经察觉到暗影门的动静，看来，自己已经成为了众矢之的。窗外，阴云密布，狂风呼啸，仿佛预示着一场暴风雨的来临。暗影门的身影，如同鬼魅般出现在宗门之外，他们的到来，无疑给青云宗带来了前所未有的压力。
“长老，暗影门的人已经逼近了。”那名弟子脸色苍白，声音颤抖。
“暗影门？”清虚真人眉头紧皱，扫过在座长老，“他们为何而来？”
“恐怕是为了林风。”一位长老，“他们知道林风的功法来历，想要借此机会，对我们青云宗下手。”
话音刚落，便有几道目光落在林风身上，其中不乏审视和担忧。林风，他知道，自己已经成为了众矢之的。
“暗影门，哼，不过是些跳梁小丑。”一位长老冷哼一声，杀意，“待会儿，让他们知道，我们青云宗的厉害。”
，一阵急促的脚步声传来，一位身穿黑色长袍的弟子匆匆走了进来，脸上带着惊恐之色。
“长老，不好了，暗影门的人已经冲进宗门了！”
话音刚落，便见那名弟子脸色大变，指着门外，失声道：“他们……他们冲进来了！”
“什么？”在座长老脸色大变，纷纷站起身来，落在门外。
“清虚真人，我们该怎么办？”一位长老沉声问道。
“保护宗门，保护林风。”清虚真人，“无论如何，都不能让他们得逞。”
随着清虚真人的话音落下，宗门内的弟子纷纷涌出，与暗影门的人展开了一场激烈的战斗。剑光闪烁，灵力涌动，整个青云宗陷入了一片混乱。
林风站在一旁，目光坚定，心中却是思绪万千。他知道，自己已经站在了风口浪尖，但无论如何，他都要坚持自己的道路，用实力证明自己的价值。
，一道身影突然出现在林风面前，正是苏雪晴。她手持一柄长剑，剑身闪烁着寒光，脸上带着坚定之色。
“林风，我们一起去战斗。”苏雪晴，“我们不能让暗影门得逞。”
林风点了点头，与苏雪晴并肩作战。他们两人，如同两道闪电，穿梭在战场之中，所向披靡。
，清虚真人与几位长老也加入了战斗。他们凭借着深厚的修为，将暗影门的人打得节节败退。
“清虚真人，你没事吧？”林风担忧地问道。
“我没事。”清虚真人，“只要我们团结一心，就一定能战胜他们。”
，一道身影突然出现在林风面前，正是王磊。他手中握着一把利剑，阴狠。
“林风，你没想到吧，我会在这里等你。”王磊冷笑道，“今天，我要让你知道，我才是真正的天才。”
话音刚落，王磊便挥剑向林风刺来。林风，迅速躲过，手中长剑瞬间挥出，与王磊展开了激烈的交锋。
“林风，你没想到吧，我会在这里等你。”王磊冷笑道，“今天，我要让你知道，我才是真正的天才。”
话音刚落，王磊便挥剑向林风刺来。林风，迅速躲过，手中长剑瞬间挥出，与王磊展开了激烈的交锋。
“林风，你的实力果然不错。”王磊冷笑道，“不过，你永远也比不上我。”
“是吗？”林风冷光，“那就让我们来试试看。”
