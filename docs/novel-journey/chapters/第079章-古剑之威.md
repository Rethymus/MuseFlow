# 第79章 古剑之威

藏经阁前，苏雪晴身姿挺拔，手持一柄古朴无华的长剑，剑尖直指暗影门刺客。她眼中寒光闪烁，剑身轻颤，仿佛蕴含着无尽的威严。
“大师姐，小心！”林风飞身而来，手中长剑舞动如风，与刺客展开激烈的交锋。剑光闪烁，暗影门刺客节节败退，却始终无法突破林风的防线。
苏雪晴，紧紧盯着刺客手中的匕首。她知道，这匕首上附着剧毒，一旦被刺中，后果不堪设想。她，体内真气涌动，古剑在手中发出低沉的鸣响，一股强大的剑气自剑身涌出，直冲云霄。
“大师姐，你没事吧？”林风见苏雪晴气息微弱，心中担忧不已。
“我没事。”苏雪晴，坚定，“林风，你退后，我来对付他们。”
林风犹豫了一下，最终还是退到了一旁。他看着苏雪晴，眼中充满了信任和敬意。
苏雪晴，体内真气如潮水般涌出，古剑在手中发出更加剧烈的鸣响。她闭上双眼，感受着古剑中的传承之力，一股强大的力量在她体内涌动。
“破！”苏雪晴猛地睁开双眼，古剑瞬间化作一道流光，直冲刺客而去。刺客还未反应过来，便被古剑一分为二，化作两截尸体，掉落在地。
，山崩地裂。苏雪晴的古剑在传承觉醒后，展现出远超凡器的威力，一剑之下可裂山断流。暗影门刺客们惊恐，纷纷后退，再也不敢靠近。
“大师姐，你……你……”林风看着苏雪晴，眼中充满了震惊和敬佩。
苏雪晴，将古剑收回体内，气息逐渐平稳。她知道，这只是传承之力觉醒的开始，她还有更长的路要走。
“林风，我们走吧。”苏雪晴转身，朝着藏经阁走去。林风紧跟其后，心中充满了疑惑。
暗影门刺客们见状，纷纷后退，不敢再靠近。他们知道，苏雪晴的古剑已经觉醒，他们无法战胜。
，暗影门刺客们的撤退，并没有让苏雪晴放松警惕。她知道，暗影门背后的势力远比她想象的要强大，他们的目的并非攻破青云宗，而是逼迫她全力催动传承之力，以便上界势力锁定她的位置。
苏雪晴警惕，她知道，她必须尽快提升自己的实力，才能应对即将到来的危机。
藏经阁内，苏雪晴盘膝而坐，开始修炼。她将古剑中的传承之力融入体内，感受着那股强大的力量在体内涌动。
“大师姐，你没事吧？”林风走了进来，看着苏雪晴，眼中充满了担忧。
“我没事。”苏雪晴睁开双眼，坚定，“林风，你不用担心，我会没事的。”
林风，下定决心，一定要保护好苏雪晴。
苏雪晴闭目修炼，体内真气涌动，古剑中的传承之力逐渐融入她的身体。她知道，她必须尽快掌握这股力量，才能应对即将到来的危机。
藏经阁外，暗影门刺客们聚集在一起，商讨着下一步的计划。他们知道，苏雪晴的古剑已经觉醒，他们无法战胜。
“头领，我们该怎么办？”一名刺客问道。
“我们必须尽快找到上界势力的支持，否则我们无法战胜苏雪晴。”头领。
“可是，上界势力对我们并不友好，他们为什么要帮助我们？”另一名刺客疑惑地问道。
“这是一个秘密，你们不必知道。”头领冷声道，“只要我们完成这个任务，上界势力自然会给我们好处。”
刺客们纷纷点头，心中充满了疑惑，但他们知道，他们必须服从头领的命令。
藏经阁内，苏雪晴修炼完毕，睁开双眼，坚定。她知道，她必须尽快提升自己的实力，才能应对即将到来的危机。

“我没事。”苏雪晴，坚定，“林风，你不用担心，我会没事的。”
林风，下定决心，一定要保护好苏雪晴。
苏雪晴起身，走向藏经阁外。她知道，她必须尽快提升自己的实力，才能应对即将到来的危机。
藏经阁外，暗影门刺客们聚集在一起，商讨着下一步的计划。他们知道，苏雪晴的古剑已经觉醒，他们无法战胜。
“头领，我们该怎么办？”一名刺客问道。
“我们必须尽快找到上界势力的支持，否则我们无法战胜苏雪晴。”头领。
“可是，上界势力对我们并不友好，他们为什么要帮助我们？”另一名刺客疑惑地问道。
