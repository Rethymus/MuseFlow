# 第99章 天衡重铸

血腥味混着焦糊气在空气中弥漫。林风握着那枚暗红色的天衡盘碎片，指尖微微发颤。碎片边缘残留着暗影门使者的血，正一点点被这块古物吸收。他低头，看见自己手腕上多了一道黑气，像活物般向上蔓延。
"林风！"苏雪晴的声音从十丈外传来。她半跪在地，左手按着右肩，鲜血从指缝间渗出。白灵在她脚边蜷缩，通体毛发倒竖，发出低沉的呜咽。
赵天磊站在她身前，长剑横在胸前，剑身上布满蛛网般的裂痕。对面，三个黑袍人悬浮于半空，兜帽下的眼睛闪烁着红光。他们手中的法器嗡嗡作响，不断吸食着周围的光线。
"碎...碎了..."林风喃喃自语。完整的天衡盘已经被分成三块，暗影门夺走了一块，苏雪晴手中一块，他手中这块最小，却隐隐跳动着与他心跳相同的节奏。
黑袍人中为首者发出刺耳的笑声："无知凡人，天衡盘已毁，你们还能如何？"
苏雪晴咬着牙站起，血从她唇角滑落："天衡盘有灵，不会轻易陨落。"
林风突然感到手中的碎片发烫。一股刺痛从掌心直冲眉心，眼前的景象开始扭曲。他看到无数画面在脑海中闪现——远古的战争，破碎的天空，一个模糊的身影手持天衡盘站在天地之间。
"那是...清虚师尊..."林风艰难地说。
碎片突然爆发出刺眼的光芒，将周围的暗影逼退。光柱中，一个虚幻的清虚真人显现，他看着林风，嘴唇无声地动着。
"他在说什么？"赵天磊惊道。
"他说...重铸..."林风的眼睛变得明亮，"但需要...代价。"
苏雪晴走近，她的伤口在光芒中迅速愈合。她伸出手，掌心浮现出一柄古朴的小剑，剑身上符文流转："这是师尊传承给我的古剑之力。"
黑袍首领怒吼："拖延时间！杀他们！"
三道黑影如箭般射来。赵天磊迎上前去，剑身上裂痕蔓延，却依然挡在苏雪晴身前。林风感到一股力量从碎片涌入体内，化神期的修为如潮水般涌动，但他知道，这还不够。
"白灵！"他喊道。
白灵化作一道白光扑向一个黑袍人，幻化出千百只狐影迷惑对手。但很快，白灵就被一道黑光击中，倒地抽搐。
"林风，我们得..."苏雪晴话未说完，突然僵住。她的眼睛直直盯着前方，仿佛看到了什么可怕的东西。
"怎么了？"林风转身，瞳孔骤缩。
在他们身后，空间扭曲，一个巨大的裂口正在形成。裂口深处，传来阵阵的低吼，无数黑影在其中若隐若现。
"是上界..."赵天磊的声音发颤，"封印...要破了..."
黑袍首领狂笑："天衡盘已毁，封印自开！待我等主人降临，尔等蝼蚁...啊！"
一道剑光突然从他背后刺入，穿透心脏。他难以置信地低头，看到苏雪晴站在他身后，古剑收回，剑尖滴血。
"你..."黑袍首领倒下，另外两人惊恐后退。
"来不及了..."林风望着那不断扩大的空间裂口，"天衡盘必须重铸。"
苏雪晴点头，她将古剑放在地上，双手结印："我以传承之力为炉。"
林风，将碎片放在古剑之上。碎片开始与古剑共鸣，发出嗡鸣。周围的空气变得粘稠，灵气疯狂向这里汇聚。
"林风，你..."赵天磊想说什么，却被林风打断。
"这是我必须做的。"林风闭上眼，化神期的修为如决堤洪水般涌入碎片。他的身体开始变得透明，仿佛随时会消散。
苏雪晴脸色苍白，额头上渗出细汗："我...撑不住了..."
白灵挣扎着爬起来，化作一道白光融入碎片，为它提供额外的灵力。
空间裂口不断扩大，几只狰狞的怪物从中探出头，发出刺耳的尖啸。赵天磊挥剑迎上，却发现自己每斩出一剑，裂口就扩大一分。
"重铸需要时间..."林风的声音变得飘忽，"但封印等不了..."
他感到自己的生命力正在被抽干，意识渐渐模糊。苏雪晴突然抓住他的手，一股温暖的力量从她体内涌来。
"我剑修血脉...与古剑共鸣..."她咬着牙，"我们一起..."
林风睁开眼，看到苏雪晴的决绝。她的身体开始发光，与林风融为一体。碎片上的光芒大盛，形成一个完美的圆盘。
"就是现在！"林风怒吼一声，将的力量注入天衡盘。
轰！
天衡盘爆发出耀眼的光芒，形成一个巨大的光罩，将空间裂口暂时封住。光罩外，那些怪物发出不甘的嘶吼，但无法突破。
光罩内，林风和苏雪晴双双倒地。赵天磊冲过来扶起他们，惊道："你们..."
林风看着手中的天衡盘，它已经恢复了完整，但表面多了一道道金色纹路，仿佛活了过来。它微微震动，指向天空某个方向。
"天衡盘..."苏雪晴虚弱地说，"它有了新的意志。"
林风抬头，看到天空中的裂口在金色纹路的照耀下开始缩小。那些怪物似乎惧怕这光芒，纷纷退回裂口深处。
"它不仅恢复了封印..."林风感受着天衡盘的变化，"它还...更强大了。"
白灵在他们身边醒来，毛发上还残留着战斗的痕迹。它凑到天衡盘前，好奇地嗅了嗅，然后发出一声满足的轻鸣。
赵天磊看着这一切，喃喃道："这...这是怎么回事？"
林风摇摇头，站起身来。虽然虚弱，但他感到一股前所未有的力量在体内流动。天衡盘仿佛与他有了某种联系，他能感受到它的心跳，与自己的心跳同步。
"清虚师尊牺牲了..."林风看着手中的天衡盘，"但他将一部分意志留在了这里。"
苏雪晴站起身，虽然脸色依然苍白，但眼中重新燃起光芒："天衡盘重铸了，而且...它获得了守护凡间的全新意志。"
天空中的裂口已经完全闭合，只留下淡淡的金色纹路在空中闪烁，仿佛一张无形的大网笼罩着青云宗。
"但暗影门还在..."赵天磊警惕地环顾四周。
林风握紧天衡盘，感受着其中流转的力量："他们还会再来。不过现在，我们有更强的力量了。"
天衡盘突然发出一声清脆的鸣响，金光一闪，没入林风的体内。林风浑身一震，感到自己与天衡盘的联系更加紧密了。
"它..."苏雪晴惊讶地看着林风，"它选择了你。"
林风闭上眼睛，感受着体内的变化。他能感觉到天衡盘的每一丝变化，能感受到它的意志。他能感觉到清虚师尊残留的一丝气息。
"师尊..."林风轻声呼唤，"我一定会完成您的遗愿。"
白灵蹭了蹭他的腿，眼中闪烁着智慧的光芒。它似乎也感受到了什么，发出一声轻柔的叫声。
"我们得回宗门了。"赵天磊说，"清虚师尊的事...需要告诉其他人。"
苏雪晴点头，但她的目光一直停留在林风身上："你没事吧？"
林风睁开眼，眼中闪烁着坚定的光芒："我很好。而且...我感觉自己变得更强了。"
他伸出手，掌心浮现出一道金色符文，那是天衡盘的力量。符文在他手中盘旋，最终消散在空气中。
"天衡盘不仅恢复了平衡通道的能力..."林风感受着体内的力量流动，"它还获得了守护凡间的全新意志。"
天空中的金色纹路越来越亮，形成一道道符文，编织成一张巨大的网，笼罩在整个青云宗上空。
"通道被永久加固了..."苏雪晴惊讶地看着天空，"这意味着上界的威胁...被彻底阻断了？"
林风摇摇头："不，只是被暂时阻断了。暗影门不会放弃，他们还会寻找其他方法。"
他看向远处，那里是后山禁地的方向。清虚师尊的牺牲，天衡盘的重铸，这一切都指向了一个更大的秘密。
"我们还有两块碎片..."林风轻声说，"必须找到它们。"
苏雪晴点头："暗影门会先我们一步。"
赵天磊握紧长剑："那还等什么？我们走！"
林风却摇了摇头："不，我们需要计划。天衡盘刚刚重铸，需要时间适应。而且..."
他感受着体内的力量，："我感觉到了，暗影门的人正在向这里聚集。"
白灵突然警觉地竖起耳朵，发出低沉的呜咽。
"他们来了..."林风说，"比我们想象的更快。"
天空中的金色纹路开始波动，仿佛在回应他的话。天衡盘的力量在体内流转，提醒着他即将到来的战斗。
"看来，我们得先解决眼前的麻烦了。"林风握紧拳头，金色的光芒在他掌心闪烁。
苏雪晴站在他身边，古剑在手，眼中闪烁着坚定的光芒："我们一起。"
赵天磊，长剑直指前方："杀。"
三人的身影在金色光芒的映照下，如同一把出鞘的利剑，直指即将到来的黑暗。天衡盘的力量在林风的体内激荡，如同一股澎湃的海潮，涌动在他的血脉之中。他感受着那份强大的力量，却同时也感到了压力，那是一种来自于责任的重量，压得他几乎无法呼吸。
“林风，你怎么样？”苏雪晴担忧地望着他，她的眼神中充满了关切。
林风，调整了一下呼吸，然后摇头：“我没事，只是...这股力量太过强大，我需要时间适应。”
赵天磊看着林风，担忧，但更多的却是坚定：“没关系，我们会在这里陪着你，直到你适应为止。”
，远处的天空突然传来了阵阵轰鸣声，仿佛是某种巨兽在咆哮。林风，他感受到了那股危险的气息。
“来了...”林风。
赵天磊握紧长剑，了冷冽的笑容：“看来，是时候给他们一个教训了。”
苏雪晴也紧握古剑，她的眼神坚定，仿佛在说，无论遇到什么样的敌人，她都不会退缩。
林风点了点头，他的眼神中闪烁着坚定的光芒：“没错，让他们见识一下我们青云宗的实力。”
话音刚落，一道道身影从天际掠过，直奔青云宗而来。那些身影高大威猛，身披黑色战甲，他们的眼神凶狠，手中握着各种法器和武器。
“暗影门！”林风低声喝道。
苏雪晴和赵天磊同时出手，他们的身影化作一道道闪电，直冲那些黑衣人而去。剑光如练，法术横飞，瞬间在空中形成一片混乱。
林风站在原地，他握紧天衡盘，感受着其中涌动的力量。他，然后猛地一挥，一道金色的光芒从天衡盘中飞出，瞬间将那些黑衣人笼罩在内。
“什么！”那些黑衣人惊呼一声，他们的身体开始颤抖，仿佛遇到了某种可怕的力量。
林风冷光，他：“天衡盘的力量，你们感受一下。”
那些黑衣人发出阵阵惨叫，他们的身体开始崩溃，化为一阵黑烟消散在空中。
赵天磊和苏雪晴看着林风，他们的眼神中充满了震惊和敬畏。
“你...你竟然能够使用天衡盘的力量！”苏雪晴惊讶地说道。
林风点了点头，他的眼神中充满了坚定：“是的，我是天衡盘的传人。”

“你...你竟然杀了我！”那长老瞪大眼睛，看着林风。
林风没有回答，他只是紧紧地握住天衡盘，感受着其中的力量。
