# 第75章 血战南门

血腥味在空气中弥漫，混合着硫磺与焦土的气息。林风的长剑泛着青光，剑尖滴落的鲜血在他脚边汇成细流。南大门的石阶上，躺着三具暗影门弟子的尸体，他们的眼睛圆睁，凝固着的惊骇。
"林师弟，左翼不稳！"赵天磊的声音从远处传来，带着一丝喘息。他的长剑已经布满缺口，衣袍撕裂处露出几道血痕。
林风没有回头，元婴期的感知让他早已察觉战局变化。他身形一晃，剑如游龙，三名偷袭的暗影门弟子咽喉同时出现细线般的血痕。倒下前，他们甚至没能发出一声惊呼。
白灵的身影在林风肩头若隐若现，雪白的毛发沾染了血点，双眼却闪烁着智慧的光芒。她轻轻一甩头，一道幻术如雾般散开，几名正要包抄的敌人突然停下脚步，茫然四顾。
"幻术师？"暗影门一名金丹后期的首领皱眉，手中黑色长刀劈向幻术源头。刀气未至，白灵已化作白光消失，出现在林风身后。
"王长老别来无恙。"林风的声音冰冷，剑尖直指对面黑袍老者。正是多次与他作对的内门弟子王磊的叔父，暗影门长老王烈。
王烈冷笑："林风，今日就是你的死期！暗影门三千精锐已将此地团团围住，插翅难飞。"他身后，数十名黑衣修士形成严密的包围圈，每个人眼中都闪烁着嗜血的光芒。
林风，元婴期的灵力在体内奔涌。他忽然想起了清虚真人的教诲："剑修之道，不在招式之多，而在一剑之纯。"手腕轻转，剑身发出一声清越的鸣响。
"剑意·破妄！"
一道剑气如闪电般射出，穿透两名挡路的金丹初期修士，直奔王烈。王烈面色一变，挥刀格挡，却被剑气震退三步，虎口崩裂。
"有点意思。"王烈舔了舔嘴唇，"可惜，你遇到的是我暗影门的《幽冥刀法》。"
他手中黑色长刀突然冒出黑色火焰，刀势一变，周围的空气都变得阴冷起来。林风感到一股寒意从心底升起，仿佛被无数双眼睛注视着。
"小心，这刀法能侵蚀心神。"赵天磊提醒道，他的剑光与林风的剑气交错，形成一道防御屏障。
林风点头，剑意再变。这一次，剑气中融入了白灵幻术的精髓。虚实结合的剑气在战场上划过，所到之处，敌人心神恍惚，动作迟滞。
"幻剑术？"王烈惊讶，随即化为贪婪，"这等功法，更适合由我暗影门来继承。"
战场另一侧，苏雪晴正指挥着炼丹房的弟子们。丹炉排成一列，炉火熊熊，药香与硝烟味混合在一起。她的额头上渗出细汗，素白的衣袍早已染上污渍，但双手依旧稳定如初。
"丹药制成，立刻送往南门！"她将一瓶回元丹递给一名弟子，"注意路线，避开密集交战区。"
弟子点头，接过丹药，身影化作一道青光消失在夜色中。苏雪晴的目光望向南门的方向，担忧如潮水般涌上心头。林风他们能撑住吗？
"大师姐，又一批敌人在东侧突破了防线。"一名急促的声音传来。
苏雪晴："林师兄那边情况如何？"
"暂时还能守住，但消耗太大。白灵师妹的幻术消耗也不轻，需要休息。"
苏雪晴："启动备用丹炉，准备'凝神丹'。这批丹药能帮助集中精神，缓解幻术疲劳。"
她亲自走向最大的丹炉，双手掐诀，灵力注入炉中。丹炉发出一阵嗡鸣，炉火突然转为幽蓝色。
"大师姐，这炉火温度太高！"一名弟子惊呼。
苏雪晴面色凝重："无妨，这是'凝神丹'的必经过程。记住，丹成后必须用冰魄珠冷却，否则药性会大打折扣。"
，一名浑身是血的弟子跌跌撞撞地跑来："大...大师姐，东侧防线...失守了！"
苏雪晴，但面上依旧镇定："慌什么？传我命令，炼丹房所有弟子听令，准备防御阵法。"
她转向几名筑基期弟子："你们负责保护丹炉，其余人随我迎敌。"
弟子们领命而去，苏雪晴看了一眼丹炉，眼中闪过决然。她拿起一把短剑，冲出了炼丹房。
南门战场上，林风的剑法越来越凌厉，但敌人如同潮水般不断涌来。元婴期的灵力也在迅速消耗，他能感觉到体内的灵力如流水般逝去。
"林师弟，再这样下去不是办法。"赵天磊喘息着说道，他的剑法大开大合，每一剑都带走数名敌人性命，但也让他消耗巨大。
林风点头，目光扫过战场。突然，他的眼睛一亮："白灵，帮我一个忙。"
白灵从肩头跃下，化作一道白光，消失在人群中。片刻后，战场上突然响起一阵骚动，无数幻象出现，敌人心神大乱。
"就是现在！"林风剑意再变，"剑意·千幻！"
他的身影在战场上不断闪烁，每一次出现都带走一条生命。剑光如练，所到之处，敌人纷纷倒下。但暗影门的人数实在太多，杀之不尽。
王烈见状，冷笑一声："以为这样就能赢吗？暗影门，阵起！"
剩余的暗影门弟子突然散开，形成一个奇特的阵法。黑色雾气从阵法中升起，形成一个巨大的黑色手掌，朝林风拍下。
"不好！"赵天磊想要救援，却被几名敌人缠住。
林风只能硬抗，剑气迎上黑色手掌。剑气与黑掌相撞，发出一声巨响，林风被震飞出去，口中溢出一口鲜血。
"哈哈哈，林风，你不行了！"王烈狞笑着，举起长刀，"今日就是你的死期！"
刀光如月，朝林风当头劈下。林风勉强举起长剑，但几近枯竭，根本无法抵挡。
就在这千钧一发之际，一道青色剑光如流星般划破夜空，与王烈的刀光相撞。两股力量相撞，发出震耳欲聋的巨响，王烈被震退数步。
"谁？"王烈惊疑不定地看向剑光来源。
只见一名青衣女子手持长剑，站在林风身前。月光下，她的容颜清冷如霜，正是天剑门的柳寒烟。
"柳师妹？"林风惊讶道。
柳寒烟没有回头，冷声道："林风，你不是他的对手。退后，让我来。"
她的剑气凛冽，带着一股不屈的意志。王烈感受着剑气中的锋芒，忌惮，随即化为冷笑："天剑门的弟子？正好，一并解决！"
刀光剑影再次交错，战场上的气氛变得更加紧张。林风趁机吞下一颗回元丹，感受着缓慢恢复。他知道，这场战斗还远未结束。柳寒烟的剑光犹如一道闪电，瞬间划破了战场的黑暗。她的剑法凌厉，每一剑都充满了破敌的意志。王烈的刀光在空中划出一道道弧线，每一次挥舞都带着一股不容抗拒的威压。
"柳师妹，你何必插手？"王烈冷笑道，手中的刀光更加凶狠。
柳寒烟没有回答，她的目光紧紧地盯着王烈，手中的剑如同她的生命一般，每一次挥舞都充满了力量。
"林风，你退后，让我来。"柳寒烟的声音冷冽而坚定。
林风看着柳寒烟。他知道，柳寒烟是为了他而来的。他点了点头，退到了一边，看着柳寒烟与王烈的战斗。
赵天磊也趁机加入了战斗，他的剑法虽然不及柳寒烟，但凭借着过人的天赋和坚韧的意志，也足以与敌人抗衡。
苏雪晴在炼丹房中，看到柳寒烟加入战斗，心中的一块大石终于落地。她知道，柳寒烟的实力远在她之上，有她在，林风他们应该能够撑住。
"大师姐，你放心，我们会守护好这里的。"一名弟子走到苏雪晴身边，说道。
苏雪晴看着弟子，微笑道："我知道，你们都是最棒的。现在，我们要做的，就是保护好这些丹炉，让它们安全地送到前线。"
，一名弟子急匆匆地跑来："大师姐，东侧防线又失守了！"

"柳师妹，小心！"林风忍不住喊道。
柳寒烟没有回头，但她的剑法却变得更加凌厉。她的剑光如同一条长龙，不断地在空中划过，每一次都准确地击中王烈的要害。

"柳师妹，你的剑法真是了得。"王烈喘息着说道。
