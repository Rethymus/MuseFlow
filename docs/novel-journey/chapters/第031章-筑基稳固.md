# 第31章 筑基稳固

林风躺在床榻之上，眼前白光闪烁，身体如同一座空荡荡的容器。筑基的成功，带给他一种难以言喻的感觉，仿佛整个人被抽离，又被重新塑形。他的呼吸变得沉重而缓慢，每一吸一呼之间，都能感觉到体内的灵力在汹涌澎湃，仿佛要冲破胸膛。
“呼……”
林风深深地吸了一口气，缓缓吐出，体内的灵力如同被释放的潮水，从四肢百骸涌入丹田，又在瞬间回流，每一次的流动，都让他的心灵得到一丝安宁。
清虚真人坐在他的床边，眼神深邃如古井，他的声音低沉而有力量：“林风，筑基只是开始，你的道路还很长。金丹之路，比筑基更为凶险，你必须要以这次筑基为基石，稳固你的根基。”
林风抬起头，目光清澈而坚定：“清虚真人，弟子明白了。”
清虚真人点了点头，目光落在林风的身上，似乎想要看透他的灵魂：“你体内的灵力已经达到了筑基期的上限，但你的心，却还未稳定。修仙之路，不仅仅是修炼，更是心性的磨砺。”
林风沉默了一会儿，他心中明白，清虚真人是在提醒他，筑基成功，并不意味着一切顺利。前方等待他的，是更为艰难的考验。
，门外传来了一阵脚步声，林风的心猛地一跳，他知道是谁来了。
苏雪晴推门而入，她的手中捧着一个小小的盒子，脸上带着淡淡的微笑：“林风，这是我为你在筑基期准备的修炼心得，希望对你有所帮助。”
林风接过盒子，打开一看，里面是一叠厚厚的纸张，上面写满了密密麻麻的文字，都是苏雪晴修炼多年的心得。
他看着苏雪晴，感激地说道：“雪晴姐姐，谢谢你。”
苏雪晴，走到床边坐下：“我们是师兄弟，互相帮助是应该的。筑基期虽然艰难，但只要心中有信念，总会走过去的。”
林风点了点头，他的心中充满了感激，他知道，自己在这个宗门中，并不孤单。
，门外突然传来一阵喧哗声，紧接着是清虚真人的声音：“怎么回事？如此喧哗？”
林风，他知道，这又是一次挑战。
他站起身，整理了一下衣衫，走到门外，只见一群弟子围在宗门广场上，中间站着王磊，他的脸上带着冷笑，周围的人群窃窃私语，似乎在议论着什么。
林风走到人群外围，目光锐利如鹰，他知道自己，必须站出来。王磊冷笑一声，指着林风：“林风，你可知你背叛了青云宗，背叛了师门！”
人群中顿时炸开了锅，纷纷指指点点，有的愤怒，有的疑惑，有的则是幸灾乐祸。
林风，平静地说道：“王磊，你所说的背叛，有何证据？”
王磊脸色一滞，他从袖中拿出一张泛黄的纸条：“这是你在藏书阁偷偷查看禁地密录的证据！”
林风接过纸条，上面果然写着一些禁地密录的记载，但他并不慌张，他知道，这是王磊在陷害自己。
“王磊，这纸条是你伪造的，对吗？”林风目光如鹰，直视王磊。
王磊脸色一变，他没想到林风会如此镇定，他咬牙切齿地说道：“林风，你以为你是什么东西？我就是要让你知道，不是谁都能在青云宗混下去！”
“清虚真人，弟子并无背叛之意，请您明察。”林风转身，对着清虚真人恭敬地说道。
清虚真人皱了皱眉，他接过纸条，仔细地看了起来，他放下纸条，目光转向林风：“林风，你告诉我，你在藏书阁到底看到了什么？”
林风，将他在藏书阁所见所闻，以及那张禁地密录的来历，一五一十地告诉了清虚真人。
清虚真人听后，沉默了片刻，他抬起头，目光锐利如鹰：“林风，你说的这一切，都是真的？”
林风坚定地点了点头：“弟子不敢欺骗真人。”
清虚真人沉吟片刻，他转向王磊：“王磊，你为何伪造证据，陷害林风？”
王磊脸色一变，他没想到清虚真人会如此问自己，他支支吾吾地说道：“清虚真人，那……那是因为……”
“是因为什么？”清虚真人语气严厉，打断了他的话。
王磊一滞，他知道自己无法再逃避：“清虚真人，那是因为……我嫉妒林风的才华，我恨他能在青云宗崭露头角。”
清虚真人冷哼一声：“王磊，你身为青云宗弟子，竟如此狭隘，你可知你犯的是什么错？”
王磊低下头，不敢直视清虚真人的目光。
清虚真人转向林风：“林风，你并未背叛师门，你做得对。但你也不要轻视了王磊，他虽错，却也是被嫉妒蒙蔽了双眼。”
林风点了点头，他明白清虚真人是在提醒自己，不要因为一时的成功而骄傲。

林风感激地点了点头，他知道，自己在这个宗门中，不仅有清虚真人这样的师尊，还有苏雪晴这样的师兄。
