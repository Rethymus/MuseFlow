# 第23章 内门风波

林风站在青云宗内门的大厅中，衣衫单薄，直视着前方那高傲的身影——王磊。王磊，内门中的佼佼者，出身世家，天赋异禀，自幼便被宗门寄予厚望。他的一言一行，一举一动，都仿佛自带光环，让人望而生畏。
大厅里，弟子们窃窃私语，目光不时扫过林风，却不敢与他直视。王磊瞥见林风，冷笑，嘲讽道：“林风，你还真是会找地方，这里可是内门，可不是你这种凡人能随意来的。”
林风没有理会王磊的嘲讽，他，目光坚定：“我站在这里，是为了证明自己。”
王磊不屑地笑了：“证明自己？就凭你那凡人出身，还敢在我面前自称证明？”
话音刚落，周围弟子纷纷看向林风，同情与不屑。林风心中一阵刺痛，但他知道，自己不能就这样屈服。
“王磊，你侮辱我的父母，就不怕清虚真人责罚？”林风冷声质问。
王磊哈哈大笑，得意：“侮辱你的父母？我只是实话实说。你这种凡人，配得上他们吗？”
林风握紧拳头，几乎要忍耐不住心中的怒火。这时，苏雪晴走了过来，轻轻拉住了林风的手，柔声说道：“林风，你先别急，王磊他就是这样的人，你不必在意。”
林风看着苏雪晴，感激，但他知道，自己不能就这样算了。
，王磊又开了口：“林风，你若想证明自己，就跟我比试一场。我若输给你，从此不再为难你。”
林风惊讶，他没想到王磊会主动提出比试。但随即，他摇了摇头：“我不同意。”
王磊脸色一变，怒道：“你不同意？难道你怕了？”
林风没有回答，只是静静地看着王磊。王磊被激怒，凶光：“好，那我就让你见识一下，什么是真正的实力。”
话音刚落，王磊便朝林风冲了过去。林风迅速躲开，但王磊的速度极快，犹如闪电一般，瞬间便来到了林风面前。
“你，敢躲？”王磊冷笑一声，手中出现了一把长剑，直指林风。
林风紧张，他知道自己必须全力以赴，否则便会败在王磊手中。
“林风，你真的要跟王磊比试？”苏雪晴急切地问道。
林风没有回答，只是了手中的剑。
，一道身影出现在了两人面前。那是赵天磊，他看着林风，担忧，但很快便恢复了平静。
“林风，你要小心。”赵天磊提醒道。
林风点了点头，目光坚定地看向王磊：“我会的。”
王磊见林风答应，得意，他举起手中的长剑，朝着林风劈了过去。
林风迅速躲开，同时挥剑反击。剑光闪烁，两人瞬间便战在了一起。
林风虽然实力不足，但他却凭借着坚韧的意志和灵活的身法，与王磊拼得难解难分。
苏雪晴和赵天磊在一旁观战，他们心中都为林风捏了一把汗。他们知道，林风虽然天资平平，但他却有着一股不屈的意志，这正是他们所敬佩的。
，一道剑气突然从天而降，朝着林风劈了过去。林风心中一惊，他立刻闪躲，但剑气却已经到了他的面前。
“小心！”苏雪晴惊呼一声，立刻挡在了林风面前。
剑气击中了苏雪晴，她被震得倒退几步，嘴角流出了一丝鲜血。
“雪晴！”赵天磊惊呼一声，立刻冲了上去。
林风见状，心中一痛，他立刻冲向苏雪晴，将她扶了起来。
“林风，你……你没事吧？”苏雪晴虚弱地问道。
林风摇了摇头，将苏雪晴扶到了一旁，他看着王磊，愤怒：“王磊，你这是何意？”
王磊冷笑一声，说道：“我只是想让你知道，你永远也比不上我。”
林风握紧拳头，坚定：“我会让你知道，我是不会输给你的。”
话音刚落，林风便朝王磊冲了过去。
赵天磊见状，立刻冲了上去，想要阻止林风。但林风的速度极快，他已经冲到了王磊面前。
两人再次战在了一起，剑光闪烁，杀气冲天。周围弟子见状，纷纷退避三舍，他们知道，这是一场生死之战。
