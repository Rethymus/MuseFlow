# 第25章 练气九层

林风盘坐在洞府中，丹田内灵气如溪流般缓缓运转。窗外月光如水，洒在青石地板上，勾勒出斑驳的影子。三个月了，自从斗法台上胜了王磊，他便将自己关在这青云后山的小洞府中，日夜不息地修炼。
指尖轻触眉心，那枚上古玉简微微发烫。无名功法在体内运转，每一丝灵气都被压榨到极致。汗水顺着额角滑落，浸湿了粗布道袍。林风双眼微闭，却能清晰感受到周围灵气的流动——它们像是有生命的溪流，环绕着他，却又无法真正进入经脉。
"还差一点。"他低语，声音沙哑。
第七次冲击失败时，林风咳出一口血。那不是普通的血，而是夹杂着黑色杂质的灵力。他抹去嘴角血迹，眼神愈发坚定。筑基，这横亘在练气和金丹之间的天堑，已经困扰了他整整三个月。外门弟子的身份已经确立，内门资源虽能享用，但真正的瓶颈，依然横亘在面前。
洞府角落，一团白光缓缓升起。白灵从睡眠中苏醒，雪白的皮毛在月光下泛着银辉。它伸出爪子，轻轻碰了碰林风的脚踝。林风睁开眼，露出一个疲惫却温暖的笑容。
"白灵，帮我看着外面。"他低声道。
灵狐点点头，化作一道白光没入窗外。林风重新闭目，双手结印。这一次，他不再急于求成，而是按照无名功法一篇所述，将灵气缓缓引向丹田深处那道无形的屏障。
屏障如同一面薄冰，灵气撞击其上，发出细微的碎裂声。林风咬牙坚持，额角青筋暴起。他想起三年前那个在青云山脚挖野菜的小少年，想起第一次踏入藏经阁时触摸到那枚玉简的悸动，想起清虚真人严厉却又藏着一丝期待的眼神。
"突破吧。"他在心中呐喊。
砰！
一声轻响，屏障碎裂。林风猛地睁开眼，丹田内灵气如潮水般奔涌，比之前浑厚了十倍不止。他能够清晰地感受到，自己的五感被无限放大，甚至能听到远处山涧中水滴落下的声音。
"练气九层。"他喃喃道，喜色。
白灵突然从窗外闪回，化作人形站在他面前。那是一约莫十五六岁的少女模样，银发如瀑，双眸如星。"主人，外面有人找你。"她轻声道，"清虚师尊。"
林风一怔，连忙整理道袍，走出洞府。月光下，清虚真人负手而立，白发如雪，面容依旧冷峻，但眼神中多了一丝温和。
"突破了？"他问道。
林风躬身行礼："回师尊，弟子刚刚突破练气九层。"
清虚真人点点头，从袖中取出一枚玉简递给他："这是筑基丹的炼制方法，以及所需的灵材清单。青云宗后山禁地深处有你需要的几种，但其余的，需要你自己去寻找。"
林风接过玉简，指尖微微颤抖。筑基，意味着真正踏入修仙者的门槛，意味着不再是一个任人欺凌的外门弟子。
"师尊，这些灵材...难寻吗？"他问道。
清虚真人目光深远："每一种都有其独特之处。紫心莲生长在灵泉之畔，需要纯净的灵气环境；冰魄草只出现在极寒之地；而龙血藤，则需要百年妖兽的鲜血浇灌。"
林风握紧玉简，心中涌起一股决心："弟子一定找到所有灵材。"
清虚真人注视着他，良久才开口："记住，修仙之路，不在于天赋多高，而在于能否坚持到底。你比许多人都有这份坚持。"
话音刚落，天边突然传来一声尖锐的啸叫。林风抬头，只见一只黑影如箭般划过夜空，直奔青云宗山门方向。
"暗影门的探子。"清虚真人面色一沉，"宗门戒备森严，他们竟然能潜入这么近。"
林风立刻进入戒备状态，无名功法在体内运转，白灵已经化作一道白光，守护在他身侧。
"师尊，需要弟子帮忙吗？"他问道。
清虚真人摇摇头："这是宗门事务，你只需专注于自己的修炼。筑基之后，你的实力才能真正提升。"他顿了顿，"不过，你可以去见识一下。这对你的修炼也有好处。"
林风点点头，随着清虚真人向山门方向疾驰而去。路上，他感觉到宗门内灵气波动，显然已经有人发现了入侵者。
"暗影门最近动作频繁，恐怕有所图谋。"清虚真人低声道。
两人来到山门时，已经有数十名青云宗弟子列阵，为首的是内门长老孙不二。他手持拂尘，面色凝重地望着前方。
"师叔。"孙不二见到清虚真人，躬身行礼。
清虚真人点点头："情况如何？"
"暗影门三名修士，筑基初期，已经突破了外围禁制。"孙不二指向远处，"弟子们正在拖延时间，等您来决断。"
林风站在清虚真人身后，目光锁定那三名黑衣人。他们气息阴冷，与青云宗弟子的灵气截然不同。为首那人手持黑色长剑，剑身上符文流转，一看就不是凡品。
"清虚真人，别来无恙啊。"黑衣人冷笑道，"三年不见，风采依旧。"
清虚真人面色不变："暗影门行事，越来越卑劣了。居然派人来我青云宗撒野。"
"卑劣？"黑衣人嗤笑，"这修仙界，本就是弱肉强食。你们青云宗独占灵脉资源，难道不怕引来天谴？"
林风注意到，黑衣人说话时，眼神不时瞟向青云宗后山方向。他心中一动，难道暗影门的真正目标是后山禁地？
"阁主何必多言。"清虚真人缓缓抬起手，"既然来了，就留下吧。"
话音刚落，天地灵气骤然变幻。清虚真人身后，青云宗弟子同时出手，数十道剑光如流星般射向三名黑衣人。
林风站在阵外，感受到剑气掠过面颊的刺痛。他运转无名功法，清晰地看到每一道剑气的轨迹，甚至能感受到黑衣人防御的薄弱之处。
"有趣的小辈。"黑衣人首领突然注意到林风，"练气后期？竟然能站在清虚真人身边？"
林风不语，紧握手中长剑。虽然境界不高，但他相信自己有自保之力。
"一起上，别浪费时间。"黑衣人首领冷喝，目标直指清虚真人。
三名黑衣人同时出手，黑色剑气如毒蛇般袭来。清虚真人拂尘轻挥，一道金色屏障瞬间展开，将剑气尽数挡下。
"孙长老，带林风回宗门。"清虚真人道，"这里不是你该来的地方。"
"师尊！"林风皱眉，"弟子可以帮忙！"
"你的实力还不够。"清虚真人语气坚定，"好好准备筑基，这才是你现在该做的事。"
林风咬了咬牙，最终听从了师尊的话。随着孙长老回到宗门，他能远远看到清虚真人独自面对三名黑衣人，金色剑气与黑色剑气交织，爆发出阵阵轰鸣。
回到自己的洞府，林风站在窗前，望着远处激战的闪光。白灵默默站在他身边，银发在风中飘动。
"主人，清虚师尊很强。"白灵轻声道。
林风点点头："但我必须尽快变强。"他握紧手中的玉简，筑基丹的灵材清单在月光下泛着微光。
清虚真人的话在他脑海中回响——修仙之路，不在于天赋多高，而在于能否坚持到底。他想起自己一路走来的艰辛，从山脚的凡人到如今的内门弟子，每一步都充满了艰难险阻。
"白灵，帮我收集一些草药。"林风突然道，"我要提前准备一些疗伤丹药，以防万一。"
白灵点点头："需要什么？"
"青云山常见的几种就行，解毒的，止血的，疗伤的。"林风道，"越多越好。"
林风取出随身携带的小药锄，走到洞府后的小药圃。这里是他亲手种植的一些基础草药，虽然品相一般，但足够应急了。他蹲下身，小心翼翼地采集着草药，动作熟练得不像一个修士，更像一个山中采药的农民。
三个月的闭关，不仅让他的修为有所突破，也让他更加专注于实用技能的学习。在藏经阁中，他不仅修炼无名功法，还阅读了大量关于炼丹、制符、阵法的书籍。虽然天赋有限，但他靠着超乎常人的毅力，已经掌握了基础炼丹术。
"这些草药不够。"林风自语，"明天去宗门药房再取一些。"
他取出一个玉瓶，将采集好的草药小心翼翼地放进去。每放一种，他都会在心中默默记下其药性和用途。这是他从苏雪晴那里学来的习惯——大师姐总是这样细致地记录每一种药材的特性。
夜深了，远处战斗的闪光已经消失。林风不知道清虚真人是否已经击退了暗影门的入侵者，但他知道，这只是开始。暗影门的频繁行动，必然有所图谋。
他坐在蒲团上，开始按照无名功法运转灵气。突破练气九层后，他的灵气更加精纯，运转速度也快了许多。丹田内的灵气如同一颗缓缓旋转的星辰，每一次转动都带起丝丝缕缕的灵力，滋养着他的经脉。
"筑基..."林风心中默念，"我一定要尽快突破。"
窗外，月光如水，洒在青云山的每一寸土地上。山中虫鸣此起彼伏，偶有灵兽经过的声响。林风沉浸修炼中，浑然不觉时间流逝。
直到第一缕晨曦透过窗棂，他才缓缓睁开眼。精光，一夜修炼，灵气又精进了些许。
"该准备筑基丹了。"他站起身，整理好道袍，"先去宗门药房，然后去后山禁地看看。"
白灵已经恢复原形，伏在他肩头。林风走出洞府，沿着青石小径向宗门方向走去。路上遇到的弟子都对他行礼，眼中带着敬意。三个月前那个默默无闻的外门弟子，如今已经成了内门中的佼佼者。
"林师兄早。"一个熟悉的声音传来。
林风转头，看到赵天磊从另一条小径走来。他看起来有些憔悴，但眼神依然锐利。
"赵师弟早。"林风点头示意。
"我听说了昨晚的事。"赵天磊压低声音，"暗影门的人真的来了？"
林风点点头："清虚师尊已经处理了。"
"那就好。"赵天磊松了口气，"最近宗门气氛有些紧张，大家都担心暗影门的报复。"
林风心中一动："你知道他们为什么会突然行动吗？"
赵天磊摇摇头："不清楚。但有人传言，他们可能是在寻找什么东西。"他顿了顿，"后山禁地最近有动静，清虚师尊加强了那里的守卫。"
林风心中一震。后山禁地？清虚真人曾经告诉过他，那里封印着重要的东西。难道暗影门的真正目标是那里？
"我也要去后山一趟。"林风道，"有些事情需要处理。"
赵天磊看了他一眼："小心点。暗影门的人狡猾得很。"
林风点点头，两人分道扬镳。来到宗门药房，苏雪晴正在整理丹药。见到林风，她露出温和的笑容。
"林师兄，需要什么丹药？"她问道。
"解毒、止血、疗伤的各十份。"林风道，"越多越好。"
苏雪晴一愣："要这么多？"
"以防万一。"林风简短解释。
苏雪晴点点头，开始准备丹药。林风站在一旁，闻着空气中弥漫的草药香气，感到一种难得的平静。
"师兄昨晚也看到了？"苏雪晴突然问道。
林风点头："远远看了几眼。"
"清虚师尊真是深不可测。"苏雪晴轻声道，"一个人对付三个筑基初期修士，还能全身而退。"
林风没有说话，但心中暗暗记下清虚真人的战斗方式。那是一种完全不同于宗门剑法的战斗风格，更加直接，也更加有效。
丹药准备好后，林风谢过苏雪晴，直奔后山禁地。一路上，他能感受到守卫比以前森严了许多。每一处都有弟子值守，灵力探测阵法遍布四周。
来到禁地入口，两名内门弟子拦住了他。
"林师兄有何贵干？"其中一人问道。
"来采集一些草药。"林风道，"清虚师尊特许的。"
两名弟子对视一眼，其中一人转身向禁地内走去。片刻后，一个身影缓步而出。
是清虚真人。
"师尊。"林风躬身行礼。
清虚真人点点头："你来了。我知道你的目的。"
林风一怔："师尊指的是？"
"筑基丹的灵材。"清虚真人道，"紫心莲和冰魄草都在禁地深处，但你不能现在去。"
"为什么？"林风不解。
"因为暗影门的人在附近活动。"清虚真人面色凝重，"他们很可能也在寻找这些灵材。"
林风握紧拳头："那更要去了。如果被他们得到，后果不堪设想。"
清虚真人注视着他，良久才开口："你有这份心，我很欣慰。但你的实力还不够。暗影门的修士不会手下留情。"
林风："弟子明白。但弟子可以自保，也能帮上忙。"
清虚真人沉默片刻，最终道："这样吧，我陪你一起去。但记住，一切听我安排。"
林风心中一喜："多谢师尊！"
清虚真人袖袍一挥，一道光罩将两人笼罩："我们直接去紫心莲生长的地方。"
光罩破开禁地的防护阵法，直接深入后山。林风能感受到周围灵气的浓郁程度比外面高出数倍，甚至能看到空气中漂浮的灵力微粒。
"前面就是紫心莲生长的地方。"清虚真人指向一处山涧，"但我们要小心，附近可能有守护兽。"
两人小心翼翼地靠近山涧，果然看到一头体型庞大的灵兽盘踞在紫心莲旁。那是一头浑身覆盖着蓝色鳞片的巨蟒，双眼如血，正警惕地扫视着四周。
"是冰晶蟒。"清虚真人低声道，"筑基后期的实力，已经有些灵智了。我们不能硬拼。"
林风运转无名功法，仔细观察着冰晶蟒的弱点。他注意到蟒腹处的鳞片相对较薄，而且它似乎对周围的声音特别敏感。
"师尊，我有个办法。"林风低声道，"我可以引开它，您趁机采摘紫心莲。"
清虚真人皱眉："太危险了。"
"我有白灵帮忙。"林风道，"而且我有把握全身而退。"
清虚真人沉思片刻，最终点头："小心点。记住，你的生命比灵材重要。"
林风点点头，示意白灵做好准备。然后他，从藏身之处走出，故意发出声响。
冰晶蟒立刻警觉起来，血红的眼睛锁定林风，口中吐出丝丝寒气。林风能感觉到周围的温度骤降，甚至能看到自己呼出的气息凝结成冰晶。
"来吧。"林风低语，身形一闪，向侧面掠去。
冰晶蟒怒吼一声，巨大的身躯如箭般射向林风。林风运转轻功，在山涧间灵活穿梭，将冰晶蟒的注意力完全吸引到自己身上。
"就是现在！"他对清虚真人喊道。
清虚真人身形如电，瞬间掠到紫心莲旁，手指轻点，将那朵散发着淡淡紫光的莲花采下。然后他转身，一道金光射向冰晶蟒的眼睛。
冰晶蟒吃痛，攻势一滞。林风抓住机会，身形如风般向远处掠去，成功将冰晶蟒引开。
"师尊，快走！"他在远处喊道。
清虚真人点点头，身形如电般向林风方向掠去。冰晶蟒似乎察觉到被骗，怒吼着追来，但已经追不上了。
"做得不错。"清虚真人赞道，"你的反应很快。"
林风喘着气："接下来去冰魄草生长的地方吗？"
清虚真人摇头："不急。先回宗门，把紫心莲处理好。冰魄草生长在极寒之地，需要更充分的准备。"
两人回到宗门，清虚真人将紫心莲交给林风："好好保存，这是筑基丹最重要的灵材之一。"
林风接过紫心莲，能感觉到其中蕴含的精纯灵力。这朵莲花看似普通，但灵气之浓郁远超他想象。
"师尊，我什么时候可以再去寻找冰魄草？"他问道。
"等你的修为再稳固一些，再等宗门局势明朗一些。"清虚真人意味深长地道，"暗影门的威胁还没有解除。"
林风点点头，他知道师尊是为了他好。但他也清楚，时间不等人。暗影门的行动越来越频繁，随时可能对宗门发动全面攻击。
"弟子明白了。"他躬身行礼，"弟子会尽快准备筑基，为宗门出一份力。"
清虚真人拍拍他的肩膀："记住，修仙之路，急不得。但也不能停。"
