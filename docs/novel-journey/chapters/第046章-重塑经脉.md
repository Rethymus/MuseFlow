# 第46章 重塑经脉

青云山巅，风卷云涌，天光透过松枝洒落，斑驳陆离。林风卧于藏经阁内的一间石室之中，四周静寂无声，只有他呼吸之声，粗重而急促。
他身上覆盖着厚厚的棉被，却依旧难以抵挡体内那如火般肆虐的痛楚。那痛楚从丹田蔓延至四肢百骸，仿佛有无数利刃在割裂他的经脉。每一次呼吸都像是在撕裂自己的灵魂。
“师尊，雪晴姐，你们在吗？”林风的喉咙干涩，声音嘶哑，带着一丝绝望。
“林风，你醒醒。”一个温柔的声音在石室中回荡，是苏雪晴。她推门而入，手中捧着一碗散发着淡淡药香的汤汁。
“这是……？”林风艰难地抬起头，眼神中闪烁着希望的光芒。
“这是师尊让我为你炼制的通脉丹，可以缓解你的痛苦。”苏雪晴将汤汁喂入林风的口中，他闭上眼睛，感受着那股清凉顺着喉咙流入丹田。
“师尊他……”林风欲言又止，眼中泪光闪烁。
“我知道你担心师尊，但他不会有事的。”苏雪晴安慰道，眼中却满是忧虑。
“雪晴姐，我……”林风想要表达自己的歉意，却发现喉咙仿佛被什么东西堵住，说不出话来。
“别说话，好好休息。”苏雪晴轻轻抚摸着林风的头发，眼中满是关切。
此刻，石室外，清虚真人站在那里，地望着石室内的林风。他轻轻摇了摇头，转身离去。
“师尊……”林风的声音再次响起，却只化作一声叹息。
石室内，林风感受到一股无形的力量在体内游走，修复着那些受损的经脉。那力量如同一把利剑，割裂着他的身体，却也在修复着他。
“这……这是什么力量？”林风惊恐地发现，那力量似乎来自他的身体，却又似乎不属于他。
“这是……无名功法？”林风心中一动，突然想起那上古玉简中记载的功法。
“难道……我这是在重塑经脉？”林风的脑海中闪过这样一个念头。
他感受到那力量在修复着他的经脉，每一次修复都伴随着剧痛。那痛楚如同烈火灼烧，让他的身体几乎无法承受。
“坚持下去，林风。”苏雪晴的声音在林风的耳边响起，带着无尽的鼓励。
林风咬紧牙关，忍受着那痛苦，他不想让苏雪晴担心，更不想让清虚真人失望。
“师尊，雪晴姐，对不起……”林风的脑海中闪过这样一个念头，他不想让他们为了自己而担忧。
“林风，你不必自责。”苏雪晴的声音再次响起，带着一丝温柔。
