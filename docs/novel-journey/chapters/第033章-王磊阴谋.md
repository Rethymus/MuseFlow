# 第33章 王磊阴谋

青云宗的山门巍峨矗立，苍松翠柏之间，灵气浓郁。林风刚从藏经阁中走出，便感受到一股熟悉的波动——是灵气，更深处，是某种微妙的变化。他停下脚步，闭上双眼，感知四周的灵气流动，心中却无端生出一抹警惕。
脚步声打破了林风的凝神，他缓缓睁开眼，只见一名年轻弟子匆匆走来，手中拿着一卷泛着微光的纸卷。林风皱眉，他知道那纸卷上的内容，那是清虚真人布置的任务，要求所有弟子在一个月内完成一项对宗门有用的修炼。
“大师兄，宗门长老召集你到聚义堂。”年轻弟子，眼神中带着一丝敬畏。
林风接过纸卷，：“我知道了。”他的语气平淡无波，却让那名弟子更加敬畏。
聚义堂内，灯火辉煌，长老们坐在堂中，面容肃穆。林风走上前去，深深一拜，朗声道：“弟子林风，见过长老。”他的声音沉稳有力，不带丝毫情绪波动。
“你筑基成功后，宗门事务颇为繁杂，今日特此召集你，是有要事商议。”一位坐在主位的苍须老者。
林风微微颔首，他清楚，这老者便是青云宗的长老之一，赵天宇，他与王磊的矛盾由来已久。林风深知，青云宗内，他不能不有所防范。
“大师兄，我听说王磊在暗中联合几名心腹，准备在门派历练中对你下手。”赵天宇忽然开口，语速极快，仿佛怕被旁人察觉。
林风心头一紧，冷光，但他仍保持着平静，淡然道：“宗门历练乃是我辈修炼之路的考验，王磊如此行事，实在是有些不妥。”
“哼！”赵天宇冷哼一声，“他不光要在历练中针对你，还暗中布局，想要借此机会打压你。”
林风眉头紧皱，他没想到王磊竟然如此心狠手辣，竟然要在宗门历练中陷害自己。他目光扫过众人，却发现，除了赵天宇之外，并没有人开口，显然是都对这件事情有所隐瞒。
“宗门历练将至，我必须小心应对。”林风，眼神中闪过一丝决然，“我会小心应对的。”
赵天宇满意地点点头，他知道林风不是那种轻易被击败的人。一名弟子匆匆走进堂内，神色惊慌，急声道：“长老，不好了！赵天磊和几名同门弟子在练武场上大打出手，似乎有人在暗中指挥。”
话音刚落，便见几道身影从练武场上奔来，其中一人正是王磊。他满脸愤怒，指着林风说道：“就是他！就是他！害得我失去筑基的资格，如今还想在我门派历练中报复！”
林风淡淡一笑，他并没有被王磊的话激怒，他知道，王磊此举正是为了转移所有人的注意力，好让他在宗门历练中顺利设局。他心中早已有数，这一次，他绝不会再让王磊得逞。
聚义堂内的长老们纷纷起身，他们知道，这一次的事情远比他们想象的严重。林风冷光，他明白，一场针对自己的阴谋，正悄悄展开。
（待续...）聚义堂内的，长老们面面相觑，眼神中透露出一丝忧虑。赵天宇站起身，：“林风，此事非同小可，你务必小心行事。”
林风微微点头，心中却。他知道，王磊此举不过是想将众人的视线从他身上转移，好让他趁机设局。他早已有所准备，决不会让王磊得逞。

“王磊，你这是何意？”赵天宇沉声问道，语气中带着一丝严厉。
“赵长老，我可是亲眼所见，林风在藏经阁中偷学秘法，导致我失去筑基的资格，他这是在报复我！”王磊指着林风，怒吼道。
林风，他并未辩解，只是静静地看着王磊，眼神中透露出一丝冷漠。
赵天宇脸色阴沉，他深知王磊所言非虚，林风的确在藏经阁中偷学秘法。他心中却有一丝疑惑，林风为何会选择这种秘法？
“大师兄，你真的在藏经阁中偷学秘法？”一名长老疑惑地问道。
林风摇了摇头，语气平静：“长老，我并未偷学秘法，只是偶然间得到了一本古籍，上面记载了筑基之法。”
“古籍？”赵天宇疑惑地问道，“那古籍从何而来？”
“是从藏经阁中偶然得到的。”林风回答道，他的眼神坚定，不容置疑。
，一名弟子匆匆走进堂内，神色焦急，急声道：“长老，不好了！王磊他们已经将林风围在了练武场中央，似乎要对他不利！”
赵天宇脸色一变，他深知王磊等人不是善茬，若让林风陷入险境，后果不堪设想。他：“立刻传令下去，让所有弟子都到练武场！”
话音刚落，便见几道身影从堂内奔出，迅速前往练武场。
练武场上，王磊等人将林风围在中央。林风，他深知，自己今日必有一场恶战。
“林风，你还不束手就擒？”王磊指着林风，冷笑道。
