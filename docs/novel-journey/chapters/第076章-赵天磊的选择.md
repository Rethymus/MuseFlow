# 第76章 赵天磊的选择

剑光如雪，寒意刺骨，赵天磊立于城头，望着南大门外的战场，心中如潮水般翻涌。暗影门的势力如同乌云般压向青云宗，战鼓擂动，喊杀声震天。他手中握着一柄长剑，剑身通体乌黑，透着一股诡异的气息。
“天磊，你可知，暗影门的背后有着怎样的势力？”清虚真人站在他的身旁，声音低沉而沙哑，仿佛自遥远的地底传来。
赵天磊微微摇头：“弟子不知，只知暗影门行事狠辣，宗门多次派遣弟子前去调查，却无一人能活着回来。”
清虚真人深深地看了他一眼，道：“这是天衡盘的力量所致，那禁地之下，隐藏着上古大能的封印，暗影门为了得到天衡盘，不惜一切代价。”
赵天磊握剑的手微微一紧，一股寒意顺着剑身蔓延至指尖。他，将心中的杂念抛开，目光如鹰隼般锐利，盯着南大门外的战局。
此时，战场上的情况愈发危急。林风如同一柄出鞘的利剑，身姿矫健，每一次挥剑都带走暗影门弟子的生命。而王磊则趁机进攻，企图抓住林风的破绽。
“林风，小心！”赵天磊高喝一声，瞬间冲入战场。他手中的剑犹如一条黑色的游龙，在空中划过一道道优美的弧线，每一次击打都精准无误。
林风回头，见到赵天磊，惊讶，但很快化作坚定的神色。他，挥剑抵挡王磊的进攻。两人配合无间，犹如一对久经沙场的战友。
战斗进行得如火如荼，赵天磊的剑法愈发犀利，他心中的杀意也愈发浓重。他想起当年与林风的过往，那时的他嫉妒林风的天赋，认为林风是自己的阻碍。在这场战斗中，他意识到，真正的敌人不是林风，而是暗影门。
“天磊，你的剑法进步了。”林风在战斗中抽空对赵天磊说道，声音中带着一丝赞赏。
赵天磊，没有说话，只是继续战斗。他知道，这场战斗不仅仅是为了宗门的安危，更是为了证明自己的价值。
苏雪晴在炼丹房内焦急地指挥着弟子们，她的脸上带着一丝苍白，但眼中却充满了坚定。她知道，这场战斗对宗门，她不能有丝毫的懈怠。
白灵站在她的身旁，静静地望着战场，雪白的毛发在阳光下闪闪发光。她与林风有着心灵相通的契约，她知道林风此刻的处境，心中充满了担忧。
柳寒烟在天剑门的首席弟子位置上，她手持一柄长剑，身姿优雅，每一次出手都精准而犀利。她与林风有着深厚的友情，她愿意为林风战斗到一刻。
此刻的战场，犹如一幅血腥的画卷。剑光交错，血肉横飞。赵天磊和林风并肩作战，他们的剑法如行云流水，每一次攻击都带着致命的威胁。
战局愈发激烈，暗影门的攻势愈发猛烈。在赵天磊和林风的默契配合下，他们如同两尊不败的战神，屹立在战场上。
“林风，你退后！”赵天磊高喝一声，瞬间冲向王磊。他手中的剑如同一条黑色的毒蛇，狠狠地咬住了王磊。
林风回头，感激，他知道赵天磊是在保护他。他挥剑击退追击的暗影门弟子，转身与赵天磊并肩作战。
此时，苏雪晴在炼丹房内紧急炼制丹药，她的手法熟练而迅速。她知道，这场战斗需要更多的丹药支援。
白灵站在她的身旁，眼中闪烁着担忧的光芒。她知道，这场战斗对林风和赵天磊来说，她愿意为他们做一切。
