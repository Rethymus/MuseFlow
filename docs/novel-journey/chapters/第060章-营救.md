# 第60章 营救

林风屏住呼吸，指尖划过冰冷石壁。禁灵阵的纹路在黑暗中泛着幽蓝光泽，像无数细小的蛇缠绕着。他闭上眼，无名功法在经脉中流转，那些纹路仿佛活了过来，在他识海中游走。
空气凝滞，每一步都踩在刀刃上。林风能听见自己的心跳声，咚、咚、咚，在空旷的通道里回荡。远处传来铁链碰撞的声音，夹杂着低沉的咒骂。
"那小子不可能闯到这里来。"
"小心为上，长老有令，任何人接近此地格杀勿论。"
林风贴着石壁移动，影子被拉得细长。拐角处，两名暗影门守卫背对而立，金丹期的气息若隐若现。他，无名功法运转到极致，身影如烟般飘过，指尖轻点两人后颈。
两具身体软软倒下，没有发出一丝声响。
通道前方，铁门紧闭。门上刻满了复杂的符文，中央镶嵌着一颗墨色晶石，正散发出阵阵寒意。林风伸手触碰，指尖传来刺骨的冰凉。
"雪晴，你在里面吗？"他低声呼唤，声音在通道中消散。
没有回应。只有晶石脉动的微光，像一只窥视的眼睛。
林风后退三步，双手结印。无名功法运转，掌心泛起淡淡青光。他猛地向前拍出，掌力击中铁门，却如泥牛入海，消失得无影无踪。
"禁灵阵..."林风皱眉，额头上渗出细汗。这门上的禁制比他想象中更为强大。
通道尽头的阴影里，传来轻微的脚步声。林风闪身躲进石壁凹处，三名黑衣人缓缓走来，胸前佩戴着暗影门的徽记。
"看守严密，那小子就算来了也是自投罗网。"一人低声道。
"长老正在举行仪式，不容有失。"另一人警惕地环顾四周，"我总觉得不对劲，最近这禁灵阵消耗的灵力..."
第三人突然停下脚步，鼻子抽了抽："血腥味？"
林风心脏猛地一缩。他屏住呼吸，无名功法运转到极致，将气息压到最低。
两名黑衣人拔出长剑，剑尖直指林风藏身的方向："谁在那里？"
没有回答。只有死一般的寂静。
"搜！"领头黑衣人厉声道，"小心有诈！"
两名黑衣人分开搜索，脚步声越来越近。林风能感觉到他们身上的杀气，如芒刺背。
就在一名黑衣人即将发现他的瞬间，林风动了。身影如鬼魅般闪出，指尖在对方咽喉上轻轻一点。黑衣人瞳孔放大，身体软软倒下。
另一名黑衣人惊觉转身，长剑如毒蛇般刺来。林风侧身避过，右手扣住对方手腕，无名功法运转，一股阴寒之力顺着手臂侵入。
"你...你是谁？"黑衣人惊恐地看着自己的手臂，皮肤以肉眼可见的速度变得灰败。
林风没有回答，手指在对方眉心一点。黑衣人身体一僵，随即倒地不起。
通道再次恢复寂静。林风站在两具尸体旁，呼吸微微急促。他知道，这只是开始。前方还有更多的守卫，还有那道无法轻易打开的铁门。
他走到铁门前，仔细观察那些符文。无名功法在识海中流转，那些原本晦涩难懂的纹路逐渐变得清晰。林风伸出手指，沿着符文的轨迹缓缓移动。
"破..."他低喝一声，指尖青光大盛。
墨色晶石开始颤抖，铁门上的符文逐一亮起。伴随着一声巨响，铁门缓缓打开，露出里面的密室。
密室中，苏雪晴被锁在石柱上，脸色苍白，嘴角带着血迹。她看到林风，惊喜，随即又化为担忧："小心，这里..."
她的话没说完，密室周围的石壁突然打开，数十名黑衣人涌出，将两人团团围住。
"林风，你果然来了。"一个阴冷的声音从人群后传来，"长老有请。"
王磊从阴影中走出，脸上带着狞笑："你以为你能救她出去？暗影门的势力岂是你能想象的？"
林风护在苏雪晴身前，无名功法运转到极致："放了她，否则你们都得死。"
"哈哈哈！"王磊大笑，"就凭你？一个练气期的废物？"
苏雪晴突然挣扎起来，锁链发出刺耳的声音："别管我，快走！"
林风没有动。他能感觉到苏雪晴体内的灵力正在被某种力量抽取，如果不及时救她，后果不堪设想。
"动手！"王磊冷喝一声，黑衣人们同时发动攻击，刀光剑影如潮水般涌向林风。
林风，身影在刀光剑影中穿梭。无名功法运转，他的动作诡异莫测，总能以不可思议的角度避开攻击，同时反击。
一名黑衣人的长剑刺向林风心口，林风侧身避开，右手扣住对方手腕，无名功法运转，阴寒之力顺着手臂侵入。黑衣人惊恐地发现，自己的手臂正在失去知觉。
"啊！"惨叫声中，黑衣人倒地不起。
另一名黑衣人从背后偷袭，长剑刺向林风后背。苏雪晴看到，眼中闪过焦急："小心！"
林风仿佛背后长眼，身形一转，避开长剑，同时反手扣住对方手腕。无名功法运转，阴寒之力再次侵入。
"这不可能..."王磊脸色微变，"他的功法是什么？"
林风没有回答，他的注意力全部集中在周围的敌人上。无名功法运转，他的动作越来越快，身影在黑衣人群中穿梭，每一次出手都带走一条生命。
"保护长老！"王磊惊恐地喊道，"不能让他靠近祭坛！"
林风听到"祭坛"二字。他不知道暗影门在搞什么鬼，但他能感觉到，苏雪晴体内的灵力正在被某种力量抽取。
"雪晴，坚持住！"林风大喊一声，身影如箭般冲向王磊。
王磊惊慌失措，转身就跑。林风紧追不舍，无名功法运转，他的速度越来越快。
"拦住他！"王磊厉声道，几名黑衣人同时扑向林风。
林风不退反进，身影从黑衣人中间穿过，同时无名功法运转，阴寒之力侵入几人体内。几名黑衣人惨叫着倒地不起。
王磊跑到祭坛前，祭坛中央有一道传送阵，正在缓缓转动。他慌忙冲进传送阵："林风，你赢不了的！暗影门的势力遍布整个修仙界！"
林风追到祭坛前，却停下了脚步。他能感觉到传送阵中传来的强大气息，不是他能对抗的。
"林风..."苏雪晴虚弱的声音传来，"别管我了...快走..."
林风转身走向苏雪晴，无名功法运转，双手按在石柱上。阴寒之力顺着手臂传入石柱，石柱上的符逐一亮起。
"锁链..."苏雪晴提醒道，"锁链上有禁制..."
林风没有说话，他的指尖在锁链上划过，无名功法运转。锁链上的符逐一亮起，然后断裂。
苏雪晴自由了，但她身体虚弱，几乎站立不住。林风扶住她，感受着她体内的灵力正在被某种力量抽取。
"他们...在抽取我的灵力..."苏雪晴虚弱地说道，"用来...打开禁制..."
林风眉头紧锁。他知道，如果不阻止暗影门的仪式，整个青云宗都将面临灾难。
"雪晴，你能感应到仪式的核心吗？"林风问道。
苏雪晴闭上眼睛，感受了一下："在...祭坛下方..."
林风点点头，抱起苏雪晴，走向祭坛。周围的黑衣人纷纷涌来，试图阻止他们。
"拦住他们！"王磊的声音从传送阵中传来，"不能让他们靠近祭坛！"
林风不理会黑衣人的攻击，抱着苏雪晴径直走向祭坛。无名功法运转，他的动作诡异莫测，总能以不可思议的角度避开攻击。
"林风...我撑不住了..."苏雪晴虚弱地说道，"我的灵力...正在被快速抽取..."
林风加快脚步，终于来到祭坛前。他放下苏雪晴，双手按在祭坛上，无名功法运转。阴寒之力顺着手臂传入祭坛，祭坛上的符逐一亮起。
"找到了！"林风惊喜地说道，"仪式的核心在这里！"
苏雪晴也感应到了，她挣扎着站起来，双手与林风一起按在祭坛上："我...我可以用灵力...破坏它..."
林风点点头，无名功法运转，阴寒之力与苏雪晴的灵力同时注入祭坛。祭坛剧烈震动，上面的符逐一熄灭。
"不！"王磊惊恐的吼声从传送阵中传来，"仪式不能被破坏！"
林风和苏雪晴继续注入力量，祭坛上的符越来越暗。突然，祭坛中央裂开一道缝隙，露出了一个黑色的球体。
"那是...天衡盘的碎片..."苏雪晴惊恐地说道，"他们想用它...破坏封印..."
林风眉头紧锁。他知道，如果让暗影门拿到天衡盘的碎片，整个修仙界都将面临灾难。
"雪晴，集中所有灵力，摧毁它！"林风说道。
苏雪晴点点头，集中所有灵力，与林风一起注入黑色球体。黑色球体剧烈震动，上面的符逐一熄灭。
"不！"王磊的吼声越来越远，"我一定会回来的！"
伴随着一声巨响，黑色球体炸裂，祭坛也随之崩塌。周围的石壁开始倒塌，整个密室都在震动。
"走！"林风抱起苏雪晴，冲向密室出口。身后的通道正在倒塌，碎石如雨般落下。
他们冲出密室，沿着通道奔跑。身后的通道越来越窄，碎石越来越多。
"快到了！"林风说道，希望。
就在他们即将冲出通道的瞬间，一块巨大的石壁挡住了去路。
"不！"林风怒吼一声，无名功法运转到极致，双手按在石壁上。阴寒之力顺着手臂传入石壁，石壁上出现了一道道裂纹。
"林风...我..."苏雪晴虚弱地说道，"我可以用剑气..."
林风点点头，放下苏雪晴。苏雪晴拔出长剑，集中所有灵力，剑气如虹，击中石壁。
伴随着一声巨响，石壁崩塌，阳光照了进来。
他们终于逃出了暗影门的据点，但前方的路还很长。暗影门的势力没有被完全铲除，王磊逃走了，天衡盘的碎片也不知所踪。
林风扶着苏雪晴，站在阳光下，感受着久违的温暖。他知道，这只是开始，还在后面。阳光透过破碎的石壁，洒在林风和苏雪晴的身上，带来了一丝久违的温暖。他们并未因此放松警惕。身后的暗影门守卫们，虽然被林风的诡异力量击败，但并未就此放弃。他们疯狂地涌向密室出口，企图将林风和苏雪晴重新捕获。
“林风，你怎么样？”苏雪晴气息微弱，但她依然关切地问道。
林风转过头，看到苏雪晴苍白的脸色，心中一阵疼痛。他摇了摇头，没有说话。他知道，现在不是关心自己的时候，他们必须尽快离开这里。
“苏雪晴，你先走，我来断后。”林风说完，转身面对着涌来的暗影门守卫。
苏雪晴想要说什么，但林风已经冲了出去。他身上的无名功法流转，身体仿佛化为一道闪电，瞬间出现在暗影门守卫的面前。他的拳头带着强烈的阴寒之力，狠狠地击在守卫的胸口。守卫发出一声惨叫，倒在地上，不再动弹。
，这只是开始。更多的暗影门守卫涌了上来，林风不断地击退他们，但他的身体也受到了严重的伤害。他的衣服已经破烂不堪，鲜血染红了双手。
“林风，你快走！”苏雪晴的声音在身后传来，带着一丝焦急。
林风停下脚步，回头望去，只见苏雪晴正艰难地向前移动。她的脸上带着痛苦的表情，显然已经到了极限。
“你先走，我会跟上你的。”林风说完，转身继续战斗。
苏雪晴看着林风的背影，心中充满了担忧。她知道，林风为了保护她，已经付出了巨大的代价。她咬紧牙关，加快了脚步。
，一道黑影从侧面袭来。林风迅速转身，避开了攻击。黑影并没有放弃，他再次发动攻击，这一次，林风没有来得及躲避。
黑影的攻击准确地击中了林风的胸口，一股强烈的疼痛从胸口传遍全身。林风倒在地上，无法动弹。
“林风！”苏雪晴的声音从远处传来，带着一丝绝望。
林风艰难地抬起头，看到苏雪晴正在向他跑来。他的心中充满了感动，他知道，自己不能放弃。
“苏雪晴，你快走！”林风用尽一丝力气喊道。
苏雪晴停下脚步，看着林风，眼中充满了泪水。她知道，林风已经到了极限，但她不能就这样放弃。
“林风，我...我陪你。”苏雪晴说完，转身再次向林风跑来。
林风看着苏雪晴的背影，心中充满了感激。他知道，只要有苏雪晴在身边，他就有了继续战斗的勇气。
，一道剑光闪过，一名暗影门守卫被击倒在地。是柳寒烟，她终于赶到了。
“林风，你怎么样？”柳寒烟关切地问道。
林风看着柳寒烟，微笑着摇了摇头。他知道，自己已经没有力气说话了。
“苏雪晴，你们快走！”林风说完，闭上了眼睛。
苏雪晴和柳寒烟看着林风，心中充满了悲痛。她们知道，林风已经尽力了。
，一道强大的气息从远处传来。是清虚真人，他终于赶到了。
“林风，你醒醒！”清虚真人说完，伸手按在林风的胸口。
一股强大的灵力从清虚真人的手中流入林风的体内，林风的身体开始逐渐恢复。
“清虚真人，谢谢你。”林风虚弱地说道。
“不用谢。”清虚真人说完，转身看向苏雪晴和柳寒烟，“你们也受了伤，跟我来。”
苏雪晴和柳寒烟点点头，跟随着清虚真人离开了密室。
林风看着她们离开的背影，心中充满了感激。他知道，只要有这些朋友在身边，他就有了面对未来的勇气。
而在这片剑道苍穹中，林风的故事。林风缓缓睁开眼睛，眼前是一片模糊的景象。他挣扎着想要坐起，但身体却像是灌了铅一般沉重。清虚真人站在他的床前，手中捏着几颗晶莹剔透的丹药。
“林风，你醒了。”清虚真人温和地说道，“你的伤势已经稳定，但还需要一段时间修养。”
林风点了点头，他感觉到体内的灵力正在缓缓恢复。他挣扎着想要起身，但身体却依然无力。
“清虚真人，苏雪晴和柳寒烟呢？”林风问道。
清虚真人，叹了口气：“她们已经离开了，去寻找天衡盘的碎片了。”
林风沉默了片刻，他知道，自己不能就这样放弃。他挣扎着想要站起来，但身体却像是被重锤击中一般。
“林风，你的伤势还未痊愈，不宜过于劳累。”清虚真人阻止道。
林风咬了咬牙，他知道自己不能拖累大家。他，试图调动体内的灵力。
“清虚真人，我想出去。”林风说道。
清虚真人皱了皱眉，他看着林风，犹豫。
“林风，你的伤势还未痊愈，出去只会加重伤势。”清虚真人说道。
林风咬了咬牙，他知道自己不能拖累大家。他，试图调动体内的灵力。
“清虚真人，我必须出去。”林风说道，“苏雪晴和柳寒烟她们需要我的帮助。”
清虚真人看着林风，坚定。他点了点头，将手中的丹药递给林风。
“好吧，既然你如此坚持，那就去吧。”清虚真人说道，“但你要记住，你的身体才是最重要的。”
林风接过丹药，他感激地看着清虚真人。他知道，自己不能辜负清虚真人的期望。
林风走出密室，眼前是一片荒芜的景象。他站在密室门口，看着远处的山脉，心中充满了决心。
“林风，你怎么样？”苏雪晴的声音从远处传来。
林风回头望去，只见苏雪晴和柳寒烟正向他走来。他们的脸上带着疲惫，但眼中却充满了坚定。
“我没事。”林风说道，“你们找到了天衡盘的碎片吗？”
苏雪晴摇了摇头，她看着林风，眼中充满了担忧。
“林风，你的伤势怎么样？”苏雪晴问道。
林风苦笑了一下，他知道自己不能瞒她们。
“我的伤势还未痊愈，但我可以尽力。”林风说道。
苏雪晴和柳寒烟对视一眼，她们知道，林风已经尽力了。
“林风，我们找到了一些线索。”苏雪晴说道，“天衡盘的碎片可能藏在这片山脉之中。”
林风点了点头，他看着远处的山脉，心中充满了决心。
“我们一起去找。”林风说道。
苏雪晴和柳寒烟点点头，她们跟着林风，向着山脉深处走去。
山脉之中，阴森恐怖，充满了危机。林风他们小心翼翼地前进，尽量避免触发陷阱。
突然，一阵强烈的从前方传来。林风他们停下脚步，他们知道，他们找到了天衡盘的碎片。
“在那里！”苏雪晴指着前方说道。
林风他们加快了脚步，他们向着传来的方向走去。
在一座破败的宫殿前，他们看到了一个巨大的石碑。石碑上刻着一行字：“天衡盘，藏于地底。”
林风他们走到石碑前，他们知道，他们必须进入宫殿，才能找到天衡盘。
宫殿之中，充满了危机。林风他们小心翼翼地前进，尽量避免触发陷阱。
在一间密室中，他们看到了一个巨大的石台。石台上，放着一块散发着奇异光芒的玉简。
“那就是天衡盘。”苏雪晴说道。
林风走上前，他拿起玉简，他感觉到一股强大的力量从玉简中传来。
“林风，你怎么样？”苏雪晴关切地问道。
林风摇了摇头，他知道自己不能在这里停下。
“我们要尽快离开这里。”林风说道。
苏雪晴和柳寒烟点点头，她们跟着林风，离开了宫殿。
林风他们离开山脉，他们知道，他们的任务还没有完成。他们必须找到天衡盘的其余碎片，才能解开天衡盘的秘密。
而在这片剑道苍穹中，林风的故事。林风紧握着手中的玉简，感受到一股强大的力量在体内涌动。他抬头望去，苏雪晴和柳寒烟也站在他的身旁，她们的眼神中充满了期待和紧张。
“林风，我们要小心。”苏雪晴提醒道，“天衡盘的力量太过强大，我们必须小心应对。”
林风点了点头，他，试图平复内心的激动。他知道，他们必须尽快离开这里，否则天衡盘的力量可能会失控。
他们小心翼翼地离开了宫殿，来到了山脉的出口。当他们走出山脉时，却发现天剑门和暗影门的弟子已经在这里等候。
“林风，你们终于出来了。”天剑门的首领道，“我们一直在等你。”
林风皱了皱眉，他看着天剑门的首领，警惕。
“你们为什么会在这里？”林风问道。
“我们自然是为了天衡盘。”天剑门的首领冷笑一声，“天衡盘的力量对于我们来说。”
林风沉默了片刻，他知道自己不能轻易相信对方。他，试图调动体内的灵力。
“你们想要天衡盘，我们可以合作。”林风说道，“但你们必须答应我，不能伤害苏雪晴和柳寒烟。”
天剑门的首领犹豫了一下，他看着林风，贪婪。
“好吧，我们可以答应你。”天剑门的首领说道，“但你要记住，天衡盘的力量对于我们来说。”
林风点了点头，他知道自己不能轻易相信对方。他，试图调动体内的灵力。
“我们走吧。”林风说道。
